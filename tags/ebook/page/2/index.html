HTML 目錄轉資料夾結構工具

這是一個 Python 應用程式，用於將 HTML 格式的目錄結構轉換為實際的資料夾層次結構，並為每個章節創建 PDF 頁面捷徑。

功能特色

  • 📁 解析 HTML 目錄結構（<ul> 和 <li> 標籤）
  • 🗂️ 自動創建對應的資料夾層次結構
  • 🔢 為每個資料夾添加層次編號前綴（如 01.00.00_）
  • 🔗 為每個章節創建 PDF 頁面捷徑（.lnk 檔案）
  • 🧹 自動清理資料夾名稱，移除 Windows 不允許的字符
  • 📖 支援多層巢狀目錄結構

專案結構

app/
├── app.py                    # 主要應用程式檔案
├── Coding-Interviews.htm     # 輸入的 HTML 目錄檔案
├── folder/                   # 生成的資料夾結構
│   ├── 00_Cover/
│   ├── 01_Chapter-1_-Interview-Process/
│   ├── 02_Chapter-2_-Programming-Languages/
│   └── ...
├── 00_issue/                 # 問題描述
├── 00_ref/                   # 參考檔案
└── README.md                 # 本說明檔案

系統需求

  • Python 3.6+
  • Windows 作業系統（用於創建 .lnk 捷徑）
  • 以下 Python 套件：
    • beautifulsoup4 - HTML 解析
    • pywin32 - Windows COM 介面

安裝步驟

  1. 確保已安裝 Python 3.6 或更高版本
  2. 安裝必要的套件：
pip install beautifulsoup4 pywin32

使用方法

  1. 將您的 HTML 目錄檔案命名為 Coding-Interviews.htm 並放在專案根目錄
  2. 在 app.py 中修改 PDF 檔案路徑（第 115 行）： 
    pdf_path = r"您的PDF檔案路徑"
  3. 執行程式： 
    python app.py

電子書目錄分割工具

這是一個用於處理電子書目錄的 Python 工具，可以將大型的目錄文件自動分割成各個章節的獨立 Markdown 文件，並進行格式化處理。

功能特色

  • 📚 自動章節分割：將大型目錄文件按章節自動分割
  • 🔧 智能格式化：自動處理標題格式、編號和空格
  • 📁 結構化組織：為每個章節創建獨立的資料夾和文件
  • 🏷️ 標準化命名：使用統一的章節編號格式（ch01, ch02…）
  • 🔄 批量處理：一次性處理整個目錄結構

項目結構

app/
├── app.py                 # 主程序文件
├── toc.md                # 原始目錄文件
├── toc_modified.md       # 格式化後的目錄文件
├── chapter/              # 章節文件輸出目錄
├── 00_issue/            # 問題記錄目錄
│   └── issue.md
├── ref/                 # 參考文件目錄
│   └── format-toc.py
└── run.bat              # Windows 批次執行文件

安裝與使用

系統需求

  • Python 3.6 或更高版本
  • Windows 系統（支援批次文件執行）

使用方法

方法一：使用批次文件（推薦）

# 在 Windows 命令提示字元中執行
run.bat toc.md

方法二：直接執行 Python 腳本

# 在專案根目錄執行
python app.py toc.md

參數說明

  • toc.md：輸入的目錄文件路徑（必需參數）

處理流程

  1. 讀取輸入文件：載入指定的目錄文件
  2. 格式化處理：
    • 將中文括號 （） 轉換為英文括號 ()
    • 處理標題層級和編號格式
    • 統一章節編號格式為 ch01_, ch02_ 等
  3. 生成格式化文件：保存為 toc_modified.md
  4. 分割章節：
    • 按 ### 標記分割章節
    • 為每個章節創建獨立資料夾
    • 生成對應的 Markdown 文件

# 多欄文字轉圖片工具說明文件

## 功能概述
這個工具可以將純文字檔案轉換成多欄排版的圖片，支援自定義欄數、字體大小和圖片高度，適合用於製作文字展示海報或資料視覺化。

## 主要功能
- 支援多欄文字排版
- 自動處理中文字體
- 跨平台字體支援（Windows/MacOS/Linux）
- 可自定義圖片尺寸和字體大小

## 技術細節

### 字體處理
程式會自動檢測並使用以下字體路徑：
- MacOS: `/System/Library/Fonts/PingFang.ttc`
- Linux: `/usr/share/fonts/truetype/droid/DroidSansFallbackFull.ttf`
- Windows: `C:\Windows\Fonts\msyh.ttc`

### 參數說明
- `text`: 輸入的文字內容
- `num_columns`: 欄數（預設值視需求而定）
- `image_height`: 圖片高度（預設 924 像素）
- `font_size`: 字體大小（預設 14 像素）
- `column_width`: 每欄寬度（固定 300 像素）

### 輸出
- 輸出檔案名稱：`!!output.jpg`
- 格式：JPEG
- 背景：白色
- 文字顏色：黑色

## 使用範例
```python
create_multicolumn_image(text, num_columns=10, image_height=800, font_size=15)

注意事項

  1. 確保輸入文字檔案使用 UTF-8 編碼
  2. 系統需要安裝對應的中文字體
  3. 文字量需要與欄數相匹配，避免排版溢出

可能的優化方向

  1. 自動調整欄寬以適應最長的文字行
  2. 添加文字對齊選項
  3. 支援更多輸出格式
  4. 添加文字顏色自定義功能
  5. 實現自動分頁功能

錯誤處理建議

  • 添加字體檔案存在性檢查
  • 添加輸入文字長度驗證
  • 處理圖片生成異常情況
