第一章 緒論

第一節 主題概述

在傳統的RPG遊戲中，加入Wii的互動方式，讓玩家不只能體驗虛擬遊戲的世界更能享受揮動肢體的快感，透過網路和世界各地的玩家來進行互動，一起在虛擬的世界中冒險!!

第二節 動機及目的

大家很瘋 Wii 的遊戲，因為它提供了一個全新的遊戲操控方式。 讓互動的方式更具真實性。但是 Wii 的遊戲雖然好玩，但購買其遊戲 和機台卻是一筆不小的花費。遊戲就是要休閒，有必要還要為金錢傷腦筋嗎? 網路上有許多免費又有趣的 flash 小遊戲，如果也能用 Wii 玩，那該有多好。所以我們決定製作數種類型的 flash 小遊戲，搭配 wii 的搖桿使用，讓人們可以藉由遊戲來休閒。對我們小組而言，能藉由此次機會去親身了解遊戲製作的流程，要如何製作出一個讓人覺得好玩的遊戲，讓我們不在只是個遊戲的使用者，得以轉變為製作者。

第二章 遊戲特色及架構

第一節 遊戲特色

\1. 體驗RPG故事:

玩家能在虛擬的故事扮演不同的角色，體驗不同的 任務、族群及虛擬世界上各種壯闊的景觀與引人入勝的環境。讓你如同置身在一個全新的世界。

\2. 融合多種遊戲方式:

一般RPG遊戲遊戲中，一個故事劇情只會搭配一種玩法，但我們將3種不一樣玩法的小遊戲融合在一個故事中，讓玩家能保持對遊戲的新鮮感。

\3. 體驗式wii遊戲:

搭配 wii 搖桿，讓玩家在玩遊戲時，更能體會角色的動作，更具真實性，彷彿自己置身於虛幻的遊戲一般。

第一節 相關網站分析

第二節 目標族群

18~36 歲的族群。這一階段的人們，往往花費一整天的時間在工作和學校上，一回到家就用電腦進行休閒或繼續工作。透過此遊戲，不但 可以感受遊戲故事、玩體感遊戲來活動筋骨，讓遊戲更佳的融入生活中!!!

第四章 目標期望

第一節 近程目標

先製作出一個單機版的RPG遊戲，能讓玩家使用鍵盤滑鼠來操作遊戲，能夠透過遊戲中的NPC去執行各種故事中的任務，搭配wii的方式去執行各種不同的任務，藉以體驗故事。每一個任務都是不同方式的遊戲方式。

第二節 遠程目標

做成網路連線版本的RPG連線遊戲，徹底改變現在網路遊戲的方式，讓網路遊戲也可以往wii遊戲發展。
