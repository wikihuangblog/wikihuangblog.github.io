todo-done-as-target

HTML 轉 PDF/Markdown 轉換工具

這是一個 Python 工具，可以將 HTML 檔案批量轉換為 PDF 和 Markdown 格式。

功能特色

  • 🔄 批量轉換 HTML 檔案為 PDF 和 Markdown 格式
  • 📁 支援目錄結構處理（最多處理 2 層子目錄）
  • 🎨 自定義 CSS 樣式，優化 PDF 輸出格式
  • 📄 支援 B5 頁面大小，最小邊距設定
  • 🚀 簡單易用的批次執行腳本

系統需求

Python 套件

pip install pdfkit html2text weasyprint markdown

外部工具

  • wkhtmltopdf: 用於 HTML 轉 PDF 轉換
    • 下載地址: https://wkhtmltopdf.org/downloads.html
    • 預設安裝路徑: C:\Program Files\wkhtmltopdf\bin\wkhtmltopdf.exe

檔案結構

app/
├── app.py          # 主要轉換程式（使用 pdfkit）
├── app2md.py       # 替代轉換程式（使用 weasyprint）
├── custom.css      # 自定義 CSS 樣式檔案
├── run.bat         # Windows 批次執行腳本
├── issue.txt       # 專案說明
├── target/         # HTML 檔案輸入目錄
├── target---1/     # 備用輸入目錄
├── target-ori/     # 原始檔案目錄
└── output_pdf/     # 輸出目錄（自動建立）

使用方法

方法一：使用批次腳本（推薦）

# 在 Windows 上直接執行
run.bat

方法二：直接執行 Python 程式

# 使用 app.py（pdfkit 版本）
python app.py target

# 使用 app2md.py（weasyprint 版本）
python app2md.py

程式說明

app.py

  • 使用 pdfkit 和 wkhtmltopdf 進行轉換
  • 支援自定義 CSS 樣式
  • 設定 B5 頁面大小，最小邊距
  • 同時產生 PDF 和 Markdown 檔案

app2md.py

  • 使用 weasyprint 進行轉換
  • 先將 HTML 轉為 Markdown，再轉為 PDF
  • 支援 A4 頁面大小
  • 內建 CSS 樣式

輸出設定

PDF 設定

  • 頁面大小: B5 (app.py) / A4 (app2md.py)
  • 邊距: 最小邊距設定
  • 字體: 自動換行，優化排版
  • 縮放: 可調整縮放比例

Markdown 設定

  • 保留連結和圖片
  • 保留文字格式
  • UTF-8 編碼

目錄處理

程式會自動處理以下目錄結構：

  • 輸入目錄下的所有 .html 檔案
  • 最多處理 2 層子目錄
  • 在輸出目錄中保持相同的目錄結構

自定義設定

修改 wkhtmltopdf 路徑

在 app.py 中修改：

path_wkhtmltopdf = r'C:\Program Files\wkhtmltopdf\bin\wkhtmltopdf.exe'

調整 PDF 選項

在 app.py 中修改 options 字典：

options = {
    'page-size': 'B5',        # 頁面大小
    'margin-top': '0mm',      # 上邊距
    'zoom': 1.0,             # 縮放比例
    # ... 其他選項
}

故障排除

常見問題

  1. wkhtmltopdf 未找到

    • 確認 wkhtmltopdf 已正確安裝
    • 檢查路徑設定是否正確

  2. 轉換失敗

    • 檢查 HTML 檔案是否完整
    • 確認檔案編碼為 UTF-8

  3. PDF 格式問題

    • 調整 CSS 樣式
    • 修改頁面大小和邊距設定
