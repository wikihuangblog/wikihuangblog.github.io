網頁內容爬取工具

這是一個用於從 HTML 清單文件中提取連結配對，並自動爬取網頁內容保存到本地資料夾的 Python 工具。

功能描述

  • 解析 output-taipower_combinedTable.html 文件中的連結配對
  • 提取每個 link-group 中的本地路徑（Loc）和線上連結（SEEK）
  • 使用 Playwright 自動化瀏覽器訪問需要登入的網頁
  • 將網頁內容保存到對應的本地資料夾中
  • 支援持久化登入狀態，避免重複登入

系統需求

  • Python 3.7+
  • 需要安裝的套件：
    • playwright
    • beautifulsoup4

安裝步驟

  1. 安裝 Python 依賴套件：
pip install playwright beautifulsoup4
  1. 安裝 Playwright 瀏覽器：
playwright install chromium

使用方法

  1. 確保 output-taipower_combinedTable.html 文件存在於專案根目錄
  2. 執行主程式：
python app.py
  1. 程式會自動開啟瀏覽器視窗，請完成登入（DeepSeek/Doubao 等）
  2. 登入完成後，回到終端機按下 Enter 繼續
  3. 程式會自動爬取所有連結並保存到對應的本地資料夾

文件結構

app/
├── app.py                              # 主程式
├── output-taipower_combinedTable.html  # 連結清單文件
├── 00_issue.txt                        # 需求說明文件
├── .playwright-user-data/              # Playwright 用戶資料夾（自動生成）
└── README.md                           # 說明文件

程式功能詳解

主要函數

  • parse_loc_and_seek_pairs(): 解析 HTML 文件，提取 Loc 和 SEEK 連結配對
  • ensure_directory(): 確保目標資料夾存在
  • derive_filename_from_url(): 從 URL 生成安全的檔案名稱
  • sanitize_to_filename(): 清理文字為安全的檔案名稱
  • fetch_and_save(): 爬取網頁內容並保存到本地

特色功能

  1. 持久化登入狀態: 使用 Playwright 的持久化上下文，登入狀態會被保存
  2. 智能檔案命名: 優先使用網頁標題作為檔案名稱，fallback 到 URL 路徑
  3. 安全字元處理: 自動清理不安全的檔案名稱字元
  4. 錯誤處理: 對每個連結的爬取都有獨立的錯誤處理

注意事項

  • 首次執行時需要手動完成登入流程
  • 確保有足夠的磁碟空間保存爬取的內容
  • 程式會自動建立不存在的資料夾
  • 如果某個連結爬取失敗，程式會繼續處理其他連結

輸出格式

爬取的網頁內容會以 HTML 格式保存，檔案名稱格式為：

  • 優先使用網頁標題：{網頁標題}.html
  • 備用方案：{URL最後一段}.html

故障排除

如果遇到問題，請檢查：

  1. 是否已正確安裝所有依賴套件
  2. 是否已安裝 Playwright 瀏覽器
  3. output-taipower_combinedTable.html 文件是否存在且格式正確
  4. 網路連線是否正常
  5. 目標網站是否需要特殊的登入驗證

授權

此專案僅供學習和研究使用。
