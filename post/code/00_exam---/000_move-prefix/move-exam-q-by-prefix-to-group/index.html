考試題目分類移動工具

這是一個用於自動將考試題目根據題目前綴分類並移動到對應資料夾的 Python 工具。

功能說明

本工具會讀取 category.txt 檔案中的分類對應表，然後掃描指定目錄中的所有資料夾，根據資料夾名稱的前兩位數字（題號）將資料夾移動到對應的課程類別資料夾中。

檔案結構

app/
├── app.py              # 主程式檔案
├── asset/
│   └── category.txt    # 題目分類對應表
└── README.md          # 說明文件

使用方法

1. 準備工作

確保您的目錄結構如下：

  • 根目錄包含 category.txt 檔案
  • 根目錄包含需要分類的題目資料夾（例如：01_題目名稱、02_題目名稱 等）

2. 執行指令

python app.py <根目錄路徑>

例如：

python app.py "E:\exam\questions"

3. 執行結果

程式會：

  1. 讀取 category.txt 檔案建立分類對應表
  2. 掃描根目錄中的所有資料夾
  3. 根據資料夾名稱的前兩位數字查找對應的課程類別
  4. 創建課程類別資料夾（如果不存在）
  5. 將題目資料夾移動到對應的課程類別資料夾中

category.txt 檔案格式

檔案格式為 Tab 分隔的文字檔，第一行為標題行（會被跳過），後續每行包含：

  • 題號（1-40）
  • 課程類別名稱
  • 關鍵字（可選）

範例：

題號	課程類別	關鍵字
1	計算機架構 (Computer Architecture)	RISC、指令集、處理器設計策略
2	記憶體系統 (Memory Systems)	存取速度、快取記憶體、主記憶體
...

支援的課程類別

目前支援以下課程類別：

  • 計算機架構 (Computer Architecture)
  • 記憶體系統 (Memory Systems)
  • 作業系統 (Operating Systems)
  • 數位邏輯設計 (Digital Logic Design)
  • 網頁設計 (Web Design)
  • 數位訊號處理 (Digital Signal Processing)
  • 計算機網路 (Computer Networks)
  • 資料結構 (Data Structures)
  • 演算法 (Algorithms)
  • 程式設計 (Programming)
  • 行銷 (Marketing)
  • 資訊安全 (Information Security)

注意事項

  1. 程式會自動為題號補零（例如：1 會變成 01）
  2. 如果目標課程類別資料夾不存在，程式會自動創建
  3. 移動操作會覆蓋同名的目標資料夾
  4. 請確保在執行前備份重要資料

系統需求

  • Python 3.x
  • 無需額外的第三方套件

錯誤處理

  • 如果未提供根目錄路徑，程式會顯示錯誤訊息並退出
  • 如果 category.txt 檔案不存在或格式錯誤，程式會拋出異常

範例執行

假設您有以下資料夾結構：

exam/
├── category.txt
├── 01_計算機架構題目/
├── 02_記憶體系統題目/
├── 03_計算機架構題目/
└── 04_作業系統題目/

執行 python app.py exam 後，資料夾會被重新組織為：

exam/
├── category.txt
├── 計算機架構 (Computer Architecture)/
│   ├── 01_計算機架構題目/
│   └── 03_計算機架構題目/
├── 記憶體系統 (Memory Systems)/
│   └── 02_記憶體系統題目/
└── 作業系統 (Operating Systems)/
    └── 04_作業系統題目/
