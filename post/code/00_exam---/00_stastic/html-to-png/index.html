HTML to PNG 轉換工具

這是一個簡單但功能強大的工具，可以將 HTML 文件批量轉換為 PNG 圖片格式。

功能特點

  • 支持批量轉換整個資料夾中的 HTML 文件
  • 自動調整截圖尺寸以適應頁面內容
  • 支持完整的頁面渲染，包括動態加載的內容
  • 無頭模式運行，不需要圖形界面

安裝需求

pip install selenium webdriver-manager

使用方法

  1. 確保已安裝所有依賴包
  2. 運行腳本並提供包含 HTML 文件的資料夾路徑：
python app.py /path/to/your/html/files

工作原理

此工具使用 Selenium WebDriver：

  1. 以無頭模式啟動 Chrome 瀏覽器
  2. 載入 HTML 文件
  3. 等待頁面完全渲染
  4. 自動調整視窗大小以適應內容
  5. 捕獲整個頁面的截圖
  6. 將截圖保存為 PNG 格式

注意事項

  • 確保系統已安裝 Chrome 瀏覽器
  • 轉換過程中請確保網絡連接穩定
  • 對於較大的 HTML 文件，可能需要更長的處理時間

錯誤處理

如果遇到以下錯誤：

  • “錯誤：未提供文件夾路徑” - 請確保在運行腳本時提供了正確的資料夾路徑
  • Chrome 驅動程序相關錯誤 - 請確保已正確安裝 Chrome 瀏覽器
