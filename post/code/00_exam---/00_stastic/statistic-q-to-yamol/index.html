統計學題目整理工具

這是一個用於整理統計學題目的 Python 工具，可以自動將 HTML 文件移動到對應的資料夾中。

功能說明

此工具會掃描當前目錄中的所有 HTML 文件，並根據文件名稱的前綴數字，將文件移動到對應的資料夾中。

工作原理

  1. 掃描當前目錄中的所有 HTML 文件
  2. 提取文件名稱的前綴數字
  3. 尋找以該數字開頭的資料夾（格式：數字_）
  4. 將 HTML 文件移動到對應的資料夾中

範例

假設您有以下文件結構：

app/
├── 1.html
├── 2.html
├── 1_第一章/
├── 2_第二章/
└── app.py

執行腳本後：

app/
├── 1_第一章/
│   └── 1.html
├── 2_第二章/
│   └── 2.html
└── app.py

使用方法

  1. 確保您的 Python 環境已安裝
  2. 將 app.py 放在包含 HTML 文件和對應資料夾的目錄中
  3. 在終端機中執行：
python app.py

系統需求

  • Python 3.x
  • 無需額外的第三方套件

注意事項

  • 確保 HTML 文件的名稱以數字開頭
  • 確保對應的資料夾名稱格式為 數字_描述
  • 腳本會直接移動文件，請確保重要文件已備份
  • 如果找不到對應的資料夾，會顯示相應的提示訊息

輸出訊息

  • 已將 [文件名] 移動到 [資料夾名] - 成功移動文件
  • 未找到與 [文件名] 匹配的資料夾 - 找不到對應的資料夾
  • 所有 HTML 檔案處理完成。 - 處理完成
