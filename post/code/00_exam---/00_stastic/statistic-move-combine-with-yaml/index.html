todo-done-as-target

統計移動合併工具 (Statistic Move Combine Tool)

這是一個用於自動化文件移動和合併的 Python 工具，可以根據數字前綴匹配源目錄和目標目錄，並將文件從源目錄移動到對應的目標目錄。

功能特點

  • 🔍 智能匹配: 根據目錄名稱中的數字前綴自動匹配源目錄和目標目錄
  • 📁 遞歸處理: 支持多層目錄結構的遞歸處理
  • 🚀 自動移動: 自動將匹配的文件和目錄從源位置移動到目標位置
  • 📝 詳細日誌: 提供詳細的操作日誌，顯示每個移動操作

系統要求

  • Python 3.6 或更高版本
  • Windows 操作系統（路徑配置為 Windows 格式）

安裝

  1. 克隆或下載此項目到本地
  2. 確保已安裝 Python 3.6+

使用方法

配置路徑

在運行腳本之前，請修改 app.py 中的路徑配置：

# 源路徑和目標路徑的基礎目錄
source_base = r"你的源目錄路徑"
target_base = r"你的目標目錄路徑"

運行腳本

python app.py

工作原理

  1. 目錄掃描: 腳本會遍歷源基礎目錄的所有子目錄
  2. 數字前綴提取: 從源目錄名稱中提取數字前綴
  3. 目標匹配: 在目標目錄中查找具有相同數字前綴的子目錄
  4. 文件移動: 將源目錄中的文件和子目錄移動到匹配的目標目錄

匹配規則

  • 源目錄名稱如 01_folder 會匹配目標目錄中名稱以 01 開頭的子目錄
  • 支持多層目錄結構的遞歸處理
  • 如果找不到匹配的目標目錄，會顯示相應的警告信息

目錄結構示例

源目錄結構:
source_base/
├── subdir1/
│   ├── 01_source/
│   └── 02_source/
└── subdir2/
    ├── 03_source/
    └── 04_source/

目標目錄結構:
target_base/
├── subdir1/
│   ├── 01_target/
│   └── 02_target/
└── subdir2/
    ├── 03_target/
    └── 04_target/

注意事項

⚠️ 重要提醒:

  • 此腳本會實際移動文件，請在運行前備份重要數據
  • 確保目標目錄有足夠的寫入權限
  • 建議先在測試環境中驗證腳本功能

輸出示例

已移動文件: D:\source\01_folder\file1.txt -> D:\target\01_folder\file1.txt
已處理目錄: D:\source\01_folder -> D:\target\01_folder
未找到與 D:\source\05_folder 匹配的目標目錄
文件移動完成

故障排除

常見問題

  1. 權限錯誤: 確保對源目錄和目標目錄都有適當的讀寫權限
  2. 路徑不存在: 檢查配置的路徑是否正確存在
  3. 編碼問題: 確保目錄名稱不包含特殊字符

調試建議

  • 在運行前檢查源目錄和目標目錄的結構
  • 查看控制台輸出的詳細日誌信息
  • 可以先在小型測試目錄上運行腳本
