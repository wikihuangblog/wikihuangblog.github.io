檔案路徑關鍵字提取與統計工具

這是一個用於從檔案路徑中提取大括號內的關鍵字，並生成關鍵字頻率統計圖表的 Python 工具。

功能特色

  • 🔍 關鍵字提取：從檔案路徑中自動提取大括號 {} 內的關鍵字
  • 📊 頻率統計：統計每個關鍵字的出現頻率
  • 🖼️ 視覺化輸出：生成包含關鍵字頻率表格的圖片檔案
  • 🔄 多關鍵字支援：支援單個檔案路徑中包含多個關鍵字的情況
  • 📝 自動命名：根據提取的關鍵字自動生成輸出檔案名稱

檔案結構

app/
├── app.py                                    # 主程式檔案
├── path-with-keyword-in-bracket.txt         # 輸入檔案（包含檔案路徑列表）
├── path-with-keyword-in-bracket-old.txt     # 舊版輸入檔案
├── todo.md                                  # 待辦事項和問題記錄
├── EaseUS_2024_10_15_10_55_59.jpg          # 範例輸出圖片
└── README.md                                # 說明文件

安裝需求

Python 套件

pip install Pillow

系統需求

  • Python 3.6+
  • Windows 系統（字體路徑設定為 Windows 字體）

使用方法

1. 準備輸入檔案

將包含檔案路徑的列表儲存為 path-with-keyword-in-bracket.txt，每行一個路徑。

2. 執行程式

python app.py

3. 查看結果

程式會生成一個以 !!00_index_{關鍵字列表}.jpg 格式命名的圖片檔案，包含：

  • 關鍵字頻率統計表格
  • 按頻率排序的關鍵字列表
  • 視覺化的表格格式

輸入格式範例

單一關鍵字

[v]04_四、請由小至大列出下述函數之複雜度：1000000,_nlogn,_n2__,_(3-2)n__,_.QQ.lvv.{algorithm}.md

多個關鍵字

[v]01_一、matrix-pointer-reverse.QQ.lvv.{matrix,pointer}.md

輸出格式

程式會生成類似以下格式的檔案名稱：

!!00_index_{algorithm_cloud_code_counter_dict_array_hand_dict_sum_dynamic_html_if_loop_matrix_pointer_matrix}.jpg

程式特色

關鍵字處理

  • 自動識別大括號 {} 內的內容
  • 支援逗號分隔的多個關鍵字
  • 自動去除關鍵字前後的空白字元
  • 統計每個關鍵字的出現頻率

視覺化功能

  • 生成 800x600 像素的圖片
  • 包含標題行（關鍵字、頻率）
  • 按頻率降序排列的資料
  • 清晰的表格網格線

檔案命名

  • 提取所有不重複的關鍵字
  • 按字母順序排序
  • 用底線連接所有關鍵字
  • 生成唯一的輸出檔案名稱

已知問題

根據 todo.md 記錄的問題：

  1. 檔案名稱過長可能導致檔案系統錯誤
  2. 需要處理多關鍵字分割的情況
  3. 字體路徑需要根據系統調整

技術細節

核心功能

  • 使用正則表達式 r'\{(.+?)\}' 提取大括號內容
  • 使用 Counter 進行頻率統計
  • 使用 PIL 庫生成圖片

字體設定

目前設定為 Windows 系統的微軟正黑體：

font = ImageFont.truetype("C:\\Windows\\Fonts\\msyh.ttc", 20)

開發者注意事項

  • 如需在其他作業系統使用，請調整字體路徑
  • 可根據需要調整圖片尺寸和表格樣式
  • 建議對輸入檔案進行格式驗證

授權

此專案為個人開發工具，供學習和研究使用。
