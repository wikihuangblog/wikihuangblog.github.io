Markdown 大綱提取工具

這是一個用於從多個 Markdown 檔案中提取標題並生成大綱的 Python 工具。該工具可以讀取多個 Markdown 檔案，提取其中的三級標題（###），並生成一個統一的目錄大綱，同時輸出為 Markdown 檔案和 PNG 圖片。

功能特色

  • 📁 從 input.txt 讀取多個 Markdown 檔案路徑
  • 📝 提取每個檔案中的三級標題（###）
  • 📋 生成統一的目錄大綱
  • 💾 輸出為 Markdown 檔案（outline.md）
  • 🖼️ 輸出為 PNG 圖片（outline.png）
  • 🎨 自動套用美觀的 CSS 樣式

系統需求

  • Python 3.6+
  • Chrome 瀏覽器
  • ChromeDriver（需要與 Chrome 版本匹配）

安裝依賴

pip install selenium markdown2

使用方法

1. 準備輸入檔案

在專案根目錄創建 input.txt 檔案，每行包含一個 Markdown 檔案的完整路徑：

D:\00_EXAM\00_dateExam\02\twport\00_target\112年度\program\112_資訊_資料結構與程式設計.markdown
D:\00_EXAM\00_dateExam\02\twport\00_target\112年度\security\112.markdown
D:\00_EXAM\00_dateExam\02\twport\00_target\112年度\article\memo.markdown

2. 執行程式

python app.py

3. 查看輸出

程式執行完成後，會生成以下檔案：

  • outline.md - 包含提取的大綱內容
  • outline.png - 大綱的視覺化圖片

檔案結構

app/
├── app.py              # 主程式檔案
├── input.txt           # 輸入檔案路徑列表
├── outline.md          # 輸出的大綱 Markdown 檔案
├── outline.png         # 輸出的大綱圖片
├── 00_issue/           # 問題記錄目錄
│   └── issue.markdown  # 問題說明檔案
└── 00_version/         # 版本記錄目錄
    └── app___original___.lnk

程式功能說明

主要函數

  • read_file_paths(input_file): 讀取輸入檔案中的路徑列表
  • extract_titles(md_file): 從 Markdown 檔案中提取三級標題
  • create_outline(file_paths): 創建統一的目錄大綱
  • save_outline(content, output_md, output_png): 保存大綱並生成圖片

輸出格式

生成的大綱格式如下：

## 檔案名稱1
### 標題1
### 標題2

## 檔案名稱2
### 標題3
### 標題4

注意事項

  1. 確保 Chrome 瀏覽器已安裝並可正常運行
  2. ChromeDriver 需要與 Chrome 版本匹配
  3. 輸入的檔案路徑必須存在且可讀取
  4. 程式會自動處理 UTF-8 編碼的檔案
  5. 生成的 HTML 檔案會自動清理

故障排除

ChromeDriver 問題

如果遇到 ChromeDriver 相關錯誤，請：

  1. 確認 Chrome 瀏覽器版本
  2. 下載對應版本的 ChromeDriver
  3. 將 ChromeDriver 加入系統 PATH 或放在專案目錄中

檔案編碼問題

如果遇到編碼錯誤，請確認：

  1. Markdown 檔案使用 UTF-8 編碼
  2. 檔案路徑不包含特殊字符

授權
