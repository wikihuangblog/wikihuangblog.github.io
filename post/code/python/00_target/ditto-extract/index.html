Ditto 剪貼簿提取工具

這是一個用於從 Ditto 剪貼簿管理軟體中提取和整理剪貼簿內容的 Python 工具。該工具可以將剪貼簿歷史記錄轉換為結構化的 Markdown 文件，並支援圖片內容的提取和處理。

功能特色

  • 📋 剪貼簿歷史提取：從 Ditto 資料庫中讀取剪貼簿歷史記錄
  • 📝 Markdown 轉換：將文字內容轉換為結構化的 Markdown 格式
  • 🖼️ 圖片處理：支援提取和轉換剪貼簿中的圖片內容
  • 📅 日期分組：按日期自動分組整理內容
  • 🔄 重複內容過濾：智能過濾重複的剪貼簿內容
  • 📁 自動檔案組織：創建按日期命名的資料夾結構
  • ⚡ 批次處理：支援大量歷史資料的批次處理

系統需求

  • Python 3.6+
  • Windows 作業系統（需要 Ditto 軟體）
  • 以下 Python 套件：
    • sqlite3（內建模組）
    • PIL (Pillow)
    • numpy
    • opencv-python（可選，用於圖片傾斜校正）
    • pywin32（用於 Windows 剪貼簿操作）

安裝依賴

pip install Pillow numpy opencv-python pywin32

檔案結構

app/
├── app.py              # 主要的提取程式
├── split-md.py         # Markdown 分割工具
├── ditto.bat          # 批次執行腳本
├── ditto_output/      # 輸出目錄
│   ├── 20250901/      # 按日期分組的資料夾
│   ├── 20250902/
│   └── ...
└── README.md          # 說明文件

使用方法

1. 基本使用

執行批次檔案來提取剪貼簿內容：

ditto.bat

或者直接執行 Python 腳本：

python app.py

2. 分割 Markdown 檔案

如果您有一個大型的 Markdown 檔案需要分割：

python split-md.py "path/to/your/markdown/file.md"

程式功能說明

app.py - 主要提取程式

  • 資料庫連接：連接到 Ditto 的 SQLite 資料庫
  • 時間戳轉換：將各種時間戳格式轉換為可讀的日期時間
  • 內容去重：智能過濾重複的剪貼簿內容
  • 圖片處理：提取並轉換 CF_DIB 格式的圖片資料
  • Markdown 生成：創建結構化的 Markdown 文件

split-md.py - Markdown 分割工具

  • 標題分割：根據 Markdown 標題（##）分割文件
  • 檔案命名：使用標題內容作為檔案名稱
  • 資料夾創建：為每個分割的內容創建對應的資料夾
  • 時間設定：根據標題中的時間設定資料夾的創建時間

Markdown 格式

# 2025-09-01 09:30:00

## 2025-09-01 09:30:00

[2025-09-01 09:30:00] 這是剪貼簿內容

![Image](images/image_1234567890.png)

## 2025-09-01 10:15:00

[2025-09-01 10:15:00] **重要內容**（重複3次以上的內容會加粗）

配置說明

資料庫路徑

預設的 Ditto 資料庫路徑為：

C:\Users\zero\AppData\Roaming\Ditto\Ditto.db

如需修改，請編輯 app.py 中的資料庫連接路徑。

輸出目錄

預設輸出目錄為 ditto_output，所有提取的內容都會保存在此目錄中。

注意事項

  1. 權限要求：程式需要讀取 Ditto 資料庫的權限
  2. 圖片格式：支援 PNG、BMP 等常見圖片格式的轉換
  3. 檔案編碼：所有輸出檔案均使用 UTF-8 編碼
  4. 重複處理：程式會自動清除已處理的記錄，避免重複提取

故障排除

常見問題

  1. 資料庫連接失敗

    • 確認 Ditto 軟體已安裝
    • 檢查資料庫路徑是否正確
    • 確認有足夠的檔案讀取權限

  2. 圖片轉換失敗

    • 確認已安裝 Pillow 套件
    • 檢查圖片資料是否完整
    • 某些特殊格式可能無法轉換

  3. 編碼問題

    • 確認系統支援 UTF-8 編碼
    • 檢查輸入檔案的編碼格式

開發者資訊
