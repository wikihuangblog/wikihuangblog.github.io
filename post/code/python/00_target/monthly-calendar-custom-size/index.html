月曆自訂大小生成器

一個使用 Python 和 Matplotlib 生成自訂大小月曆的工具，支援 A4 橫向格式輸出，適合列印使用。

功能特色

  • 🗓️ 生成單月或整年月曆
  • 📄 A4 橫向格式，適合列印
  • 🎨 可自訂網格顏色和背景顏色
  • 📱 高解析度輸出 (300 DPI)
  • 📋 支援 PNG 和 PDF 格式輸出
  • 🔧 簡潔的程式碼結構

安裝需求

確保您的系統已安裝以下 Python 套件：

pip install matplotlib

使用方法

基本使用

  1. 生成單月日曆
from app import create_calendar

# 生成 2025 年 1 月的日曆
create_calendar(2025, 1, cell_size=1, grid_color='black', block_color='white')
  1. 生成整年月曆
from app import create_year_calendar

# 生成 2025 年整年的日曆（PDF 格式）
create_year_calendar(2025, cell_size=1, grid_color='black', block_color='white')

參數說明

  • year: 年份
  • month: 月份（1-12，僅用於單月生成）
  • cell_size: 格子大小（預設為 1）
  • grid_color: 網格線顏色（預設為 ‘black’）
  • block_color: 背景顏色（預設為 ‘white’）

輸出格式

  • 單月日曆: 輸出為 calendar.png 檔案
  • 整年月曆: 輸出為 calendar_YYYY.pdf 檔案（包含 12 個月的日曆）

技術特點

  • 使用 matplotlib 進行圖形繪製
  • 支援 A4 橫向格式 (11.69 x 8.27 英吋)
  • 自動調整邊距以最大化日曆顯示區域
  • 高解析度輸出確保列印品質
  • 簡潔的網格設計，易於書寫和標記

自訂選項

您可以透過修改程式碼中的以下參數來自訂日曆外觀：

  • 字體大小：修改 fontsize 參數
  • 邊距設定：調整 plt.subplots_adjust() 參數
  • 輸出解析度：修改 dpi 參數
  • 顏色主題：更改 grid_color 和 block_color 參數

範例輸出

執行程式後，您將得到：

  • 清晰的網格線條
  • 星期標題（Sun, Mon, Tue, Wed, Thu, Fri, Sat）
  • 日期數字位於每個格子的左上角
  • 月份和年份標題位於底部

注意事項

  • 確保有足夠的磁碟空間儲存輸出檔案
  • PDF 檔案可能較大，特別是整年月曆
  • 建議使用高品質印表機以獲得最佳列印效果
