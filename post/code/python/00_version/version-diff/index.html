日曆生成器 (Calendar Generator)

一個使用 Python 和 matplotlib 生成日曆的簡單應用程式，可以生成單月日曆圖片或整年日曆 PDF 文件。

功能特色

  • 📅 生成單月日曆圖片 (PNG 格式)
  • 📚 生成整年日曆 PDF 文件
  • 🎨 可自定義網格顏色和背景顏色
  • 📄 A4 橫向格式，適合列印
  • 🔧 高解析度輸出 (300 DPI)

系統需求

  • Python 3.6+
  • matplotlib
  • calendar (Python 內建模組)

安裝依賴

pip install matplotlib

使用方法

生成單月日曆

from app import create_calendar

# 生成 2025 年 1 月的日曆
create_calendar(2025, 1, cell_size=1, grid_color='black', block_color='white')

生成整年日曆

from app import create_year_calendar

# 生成 2025 年整年日曆 PDF
create_year_calendar(2025, cell_size=1, grid_color='black', block_color='white')

參數說明

create_calendar 函數

  • year: 年份 (整數)
  • month: 月份 (1-12)
  • cell_size: 日曆格子大小 (預設: 1)
  • grid_color: 網格線顏色 (預設: ‘black’)
  • block_color: 背景顏色 (預設: ‘white’)

create_year_calendar 函數

  • year: 年份 (整數)
  • cell_size: 日曆格子大小 (預設: 1)
  • grid_color: 網格線顏色 (預設: ‘black’)
  • block_color: 背景顏色 (預設: ‘white’)

輸出文件

  • 單月日曆: calendar.png
  • 整年日曆: calendar_YYYY.pdf (例如: calendar_2025.pdf)

範例輸出

程式會生成：

  • 清晰的網格線
  • 星期標題 (Sun, Mon, Tue, Wed, Thu, Fri, Sat)
  • 日期數字 (半透明顯示)
  • 月份和年份標題

自定義選項

您可以修改以下參數來自定義日曆外觀：

# 自定義顏色
create_calendar(2025, 1, 
                grid_color='blue',      # 藍色網格線
                block_color='lightgray') # 淺灰色背景

# 自定義格子大小
create_calendar(2025, 1, cell_size=1.5)

注意事項

  • 生成的圖片為 A4 橫向格式
  • PDF 文件包含 12 個月份，每個月份一頁
  • 日期數字以半透明方式顯示，便於手寫記錄
  • 背景預設為白色，適合列印
