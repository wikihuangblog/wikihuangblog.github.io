代碼註解提取工具

這是一個用於從程式碼文件中提取註解並創建快捷方式的工具，支援多種程式語言，包括 Python、JavaScript、C++、Java、C# 和 Markdown。

功能特色

  • 🔍 多語言支援：支援 Python、JavaScript、C++、Java、C#、Markdown 等程式語言
  • 📝 註解提取：自動識別並提取代碼中的註解
  • 🔗 快捷方式創建：為每個註解創建可點擊的快捷方式，直接跳轉到對應的代碼行
  • 🌐 中文轉換：自動將簡體中文轉換為繁體中文
  • 📁 組織輸出：將提取的註解整理成大綱格式

支援的程式語言

語言 副檔名 註解格式 編輯器
Python .py # 註解 VS Code
JavaScript .js // 註解 VS Code
C++ .cpp // 註解, /* 註解 */ VS Code
Java .java // 註解 VS Code
C# .cs // 註解 VS Code
Markdown .md **標題**： VS Code

安裝需求

Python 版本 (推薦)

pip install opencc-python-reimplemented
pip install pywin32

Node.js 版本

npm install

使用方法

Python 版本

python app/app.py <輸入文件路徑>

範例：

python app/app.py example.py

Node.js 版本

node app/app.js <輸入文件路徑>

範例：

node app/app.js example.js

輸出結果

1. 代碼大綱文件

工具會在輸入文件的同目錄下創建 code.outline.txt 文件，包含提取的註解大綱：

01_初始化變數
02_計算總和
03_輸出結果

2. 快捷方式文件

在輸入文件的同目錄下創建 00_codeCommentExtract 資料夾，包含：

  • 01_初始化變數.lnk - 點擊可跳轉到第1行
  • 02_計算總和.lnk - 點擊可跳轉到第2行
  • 03_輸出結果.lnk - 點擊可跳轉到第3行

使用範例

假設有以下 Python 代碼 (example.py)：

def calculate_sum(a, b):
    total = a + b  # 計算兩個數字的總和
    return total   # 回傳計算結果

def main():
    result = calculate_sum(5, 3)  # 呼叫計算函數
    print(f"結果: {result}")      # 輸出結果

執行命令：

python app/app.py example.py

輸出結果：

  1. code.outline.txt 文件內容：
01_計算兩個數字的總和
02_回傳計算結果
03_呼叫計算函數
04_輸出結果
  1. 快捷方式文件：
    • 01_計算兩個數字的總和.lnk
    • 02_回傳計算結果.lnk
    • 03_呼叫計算函數.lnk
    • 04_輸出結果.lnk

配置選項

編輯器設定

工具預設使用 VS Code 作為編輯器。如需修改，請編輯 app.py 中的 process_file_content 方法：

def process_file_content(self, src_file_path):
    # 修改編輯器路徑
    obj['exe'] = r'C:\Program Files\JetBrains\PyCharm\bin\pycharm64.exe'

正則表達式模式

如需支援其他註解格式，可修改對應的正則表達式：

# Python 註解模式
obj['regex_pattern'] = r"^(?:.+)?(#.+)"

# JavaScript 註解模式  
obj['regex_pattern'] = r"^(?:.+)?(\/\/.+)"

常見問題

Q: 快捷方式無法正常跳轉？

A: 請確認 VS Code 已正確安裝並在系統 PATH 中，或修改編輯器路徑設定。

Q: 中文註解顯示亂碼？

A: 工具會自動將簡體中文轉換為繁體中文，如仍有問題請檢查文件編碼是否為 UTF-8。

Q: 如何支援新的程式語言？

A: 在 process_file_content 方法中添加新的副檔名和對應的正則表達式模式。
