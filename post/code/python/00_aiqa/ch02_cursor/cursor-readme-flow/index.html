Cursor README Flow

概述

這是一個自動化工具，旨在簡化項目文檔創建和 Cursor 編輯器工作流程。通過按 F1 鍵，工具會自動創建 README.md 文件並在 Cursor 編輯器中打開項目，同時自動生成項目代碼解釋文檔。

功能特色

  • 一鍵操作: 按 F1 鍵即可完成整個流程
  • 自動文檔創建: 自動生成帶有 YAML front matter 的 README.md 文件
  • Cursor 集成: 自動在 Cursor 編輯器中打開項目
  • 智能提示: 自動向 Cursor AI 發送代碼解釋請求
  • 路徑複製: 自動複製當前目錄路徑到剪貼簿

文件結構

app/
├── app.ahk          # AutoHotkey 主腳本
├── create_readme.py # Python 腳本，用於創建 README.md
├── go.bat          # 批處理文件，用於創建空 README.md
└── README.md       # 項目文檔

使用方法

前置條件

  1. 安裝 AutoHotkey
  2. 安裝 Python 3.x
  3. 安裝 Cursor 編輯器

設置步驟

  1. 將項目文件放在合適的位置
  2. 修改 app.ahk 中的路徑：
    • 更新 Python 腳本路徑（第 22 行）
    • 更新 Cursor.exe 路徑（第 31 行）

操作流程

  1. 複製目錄路徑: 在文件資源管理器中複製要打開的項目目錄路徑
  2. 按 F1 鍵: 觸發自動化流程
  3. 自動執行: 工具會自動：
    • 檢查並創建 README.md 文件（如果不存在）
    • 在 Cursor 編輯器中打開項目
    • 自動向 Cursor AI 發送代碼解釋請求

技術細節

AutoHotkey 腳本 (app.ahk)

  • F1 熱鍵綁定: 觸發整個自動化流程
  • 剪貼簿操作: 獲取目錄路徑
  • 文件檢查: 檢查 README.md 是否存在
  • Python 調用: 執行 Python 腳本創建文檔
  • Cursor 啟動: 自動打開 Cursor 編輯器
  • AI 交互: 自動發送代碼解釋請求

Python 腳本 (create_readme.py)

  • YAML Front Matter: 創建包含標題、日期、描述的元數據
  • 模板生成: 生成標準的 README.md 模板
  • UTF-8 編碼: 確保中文字符正確顯示

批處理文件 (go.bat)

  • 空文件創建: 創建空的 README.md 文件
  • UTF-8 編碼: 確保文件編碼正確

自定義配置

修改路徑

在 app.ahk 中修改以下路徑：

; Python 腳本路徑
pythonScript := "你的路徑\create_readme.py"

; Cursor 執行文件路徑
cursorExePath := "你的路徑\Cursor.exe"

自定義 README 模板

在 create_readme.py 中修改 content 變量來自定義 README.md 模板。

故障排除

常見問題

  1. F1 鍵無響應

    • 確保 AutoHotkey 腳本正在運行
    • 檢查腳本是否有語法錯誤

  2. Cursor 無法打開

    • 檢查 Cursor.exe 路徑是否正確
    • 確保 Cursor 已正確安裝

  3. Python 腳本執行失敗

    • 確保 Python 已安裝並在 PATH 中
    • 檢查 Python 腳本路徑是否正確

  4. README.md 創建失敗

    • 檢查目標目錄的寫入權限
    • 確保目錄路徑有效
