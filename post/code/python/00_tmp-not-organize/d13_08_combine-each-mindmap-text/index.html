HTML 內容提取與合併工具

這是一個用於從多個 HTML 文件中提取特定內容並合併成單一 Markdown 文件的 Python 工具。

功能描述

本工具專門設計用於處理包含程式碼內容的 HTML 文件，能夠：

  • 從指定的 HTML 文件中提取特定結構的程式碼內容
  • 將提取的內容合併成一個統一的 Markdown 文件
  • 自動處理多個文件並保持內容的結構化組織

專案結構

app/
├── app.py                    # 主程式文件
├── requirements.txt          # Python 依賴套件
├── src.txt                  # 輸入文件路徑列表
├── issue.txt                # 專案需求說明
├── output/                  # 輸出目錄
│   └── combined_content.md  # 合併後的內容文件
├── output-logic/            # 邏輯輸出目錄
├── output-ooo/              # 其他輸出目錄
└── 00_version/              # 版本歷史

安裝與使用

1. 安裝依賴套件

pip install -r requirements.txt

2. 準備輸入文件

在 src.txt 文件中列出要處理的 HTML 文件路徑，每行一個路徑：

D:\path\to\file1.html
D:\path\to\file2.html
D:\path\to\file3.html

3. 執行程式

python app.py

4. 查看結果

程式執行完成後，合併的內容將保存在 output/combined_content.md 文件中。

程式功能詳解

主要功能

  1. HTML 內容提取 (extract_content_from_html)

    • 使用 BeautifulSoup 解析 HTML 文件
    • 尋找特定的 CSS 選擇器：div.code-block-element-n1aweO div.code-area-XKXobb div.content-s77455 pre.container-ecJHMm code
    • 提取程式碼區塊中的文字內容

  2. 文件處理 (process_files)

    • 讀取 src.txt 中的文件路徑列表
    • 自動創建 output 目錄
    • 逐一處理每個 HTML 文件
    • 將提取的內容合併到單一 Markdown 文件

輸出格式

合併後的 Markdown 文件包含：

  • 每個源文件的標題（格式：#### File X: filename.html）
  • 提取的程式碼內容
  • 文件間的分隔線（---）

依賴套件

  • beautifulsoup4==4.12.2 - HTML 解析庫

使用範例

假設您有以下 HTML 文件結構：

<div class="code-block-element-n1aweO">
  <div class="code-area-XKXobb">
    <div class="content-s77455">
      <pre class="container-ecJHMm">
        <code>## 程式碼內容</code>
      </pre>
    </div>
  </div>
</div>

程式將提取 <code> 標籤中的內容並合併到輸出文件中。

注意事項

  1. 確保 src.txt 中的文件路徑正確且文件存在
  2. 程式只會提取第一個符合條件的程式碼區塊（if j>0: break）
  3. 輸出目錄會自動創建，無需手動建立
  4. 程式使用 UTF-8 編碼處理文件

錯誤處理

程式包含基本的錯誤處理機制：

  • 文件不存在時會顯示警告訊息並跳過該文件
  • HTML 解析錯誤時會顯示錯誤訊息並返回空列表
  • 程式會繼續處理其他文件，不會因單一文件錯誤而中斷

版本歷史

查看 00_version/ 目錄了解專案的版本演進歷史。
