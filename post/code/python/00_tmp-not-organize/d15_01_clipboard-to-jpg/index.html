剪貼簿圖片轉 Markdown 工具

這是一個 Python 工具，可以將剪貼簿中的圖片保存為 JPG 格式，並自動創建包含這些圖片的 Markdown 文件。

功能特色

  • 🖼️ 從剪貼簿獲取圖片並保存為 JPG 格式
  • 📝 自動創建 Markdown 文件並插入圖片
  • ⌨️ 使用快捷鍵操作（F9 保存圖片，F7 完成文件）
  • 🔄 支援連續創建多個 Markdown 文件
  • 📁 自動創建圖片目錄並組織文件

安裝需求

確保您的系統已安裝以下 Python 套件：

pip install Pillow keyboard tkinter

系統需求

  • Python 3.6 或更高版本
  • Windows 系統（keyboard 套件在 Windows 上運行最佳）

使用方法

主要應用程式 (app.py)

  1. 執行主程式：

    python app.py

  2. 在彈出的對話框中輸入 Markdown 檔案名稱

  3. 使用以下快捷鍵操作：

    • F9: 將剪貼簿中的圖片保存並添加到 Markdown 文件
    • F7: 完成當前 Markdown 文件的創建

  4. 完成一個文件後，程式會再次詢問下一個檔案名稱，可以繼續創建更多文件

簡單版本 (v1_clipboard-pic-ok.py)

如果您只需要將剪貼簿圖片保存為 JPG 文件：

python 00_version/v1_clipboard-pic-ok.py

這會將剪貼簿中的圖片保存為帶時間戳的 JPG 文件。

文件結構

app/
├── app.py                          # 主要應用程式
├── 00_version/
│   └── v1_clipboard-pic-ok.py     # 簡單版本
├── 00_issue/
│   └── issue.txt                   # 專案說明
└── README.md                       # 本文件

工作流程

  1. 啟動程式: 執行 python app.py
  2. 輸入檔案名稱: 在對話框中輸入要創建的 Markdown 檔案名稱
  3. 準備圖片: 將圖片複製到剪貼簿（例如：截圖、複製圖片等）
  4. 保存圖片: 按 F9 鍵將剪貼簿圖片保存並添加到 Markdown
  5. 重複步驟 3-4: 繼續添加更多圖片
  6. 完成文件: 按 F7 鍵完成當前 Markdown 文件的創建
  7. 創建新文件: 程式會詢問下一個檔案名稱，可以繼續創建更多文件

輸出格式

  • 圖片會保存在 images/ 目錄中
  • 圖片檔案名稱格式：{檔案名}_{時間戳}.jpg
  • Markdown 文件會包含所有圖片的引用

注意事項

  • 確保剪貼簿中有圖片內容再按 F9
  • 程式會自動處理 RGBA 格式的圖片轉換為 RGB
  • 按 F7 後會自動保存 Markdown 文件
  • 可以按取消按鈕退出程式

故障排除

常見問題

  1. “剪貼簿中沒有圖片”

    • 確保您已經複製了圖片到剪貼簿
    • 嘗試重新複製圖片

  2. 按鍵無響應

    • 確保程式視窗處於活動狀態
    • 在 Windows 系統上運行效果最佳

  3. 圖片保存失敗

    • 檢查是否有寫入權限
    • 確保磁碟空間充足

版本歷史

  • v1: 基本剪貼簿圖片保存功能
  • 當前版本: 支援 Markdown 文件創建和批量圖片處理
