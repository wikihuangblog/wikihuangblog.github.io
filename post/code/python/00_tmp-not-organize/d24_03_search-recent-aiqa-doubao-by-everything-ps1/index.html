AI 問答記錄搜尋與整理工具

這是一個使用 PowerShell 腳本自動搜尋、整理和生成 AI 問答記錄樹狀結構的工具。

功能特色

  • 🔍 使用 Everything 搜尋引擎快速搜尋 AI 問答記錄檔案
  • 📁 按日期自動分類整理搜尋結果
  • 🔗 自動提取 HTML 檔案中的 AI 對話 URL
  • 📊 生成美觀的樹狀結構 HTML 報告
  • ⚡ 支援豆包 (Doubao) 和 AIQA 兩種 AI 平台格式

專案結構

app/
├── app.ps1                    # 主要 PowerShell 腳本
├── tree-hierarchy.html        # 生成的樹狀結構報告
├── search_results_shortcuts/  # 按日期分類的搜尋結果捷徑
│   ├── 2025-04-01/
│   ├── 2025-04-03/
│   ├── 2025-04-12/
│   └── ...
└── 00_version/               # 版本備份

系統需求

  • Windows 作業系統
  • PowerShell 5.0 或更高版本
  • Everything 搜尋引擎 (ES-1.1.0.10 或更高版本)

安裝與設定

  1. 安裝 Everything 搜尋引擎

    • 下載並安裝 Everything
    • 確保 es.exe 位於指定路徑：S:\00_often\search\Everything\cmd\ES-1.1.0.10\es.exe

  2. 設定搜尋路徑

    • 編輯 app.ps1 中的 $pathLimit 變數
    • 設定您要搜尋的目標目錄路徑

  3. 調整搜尋參數

    • 修改 $searchPattern 以調整檔案搜尋模式
    • 修改 $dateRange 以設定日期範圍

使用方法

  1. 執行腳本

    .\app.ps1

  2. 查看結果

    • 搜尋結果會自動按日期分類存放在 search_results_shortcuts/ 目錄
    • 樹狀結構報告會生成為 tree-hierarchy.html

搜尋設定說明

搜尋模式

  • 檔案類型：支援 .doubao.html 和 .aiqa.*.html 格式
  • 日期範圍：預設搜尋 2025-02-01 到 2025-05-30
  • 路徑限制：可指定特定目錄進行搜尋

URL 提取規則

  • 豆包格式：提取 id="doubaoURL" 中的連結
  • AIQA 格式：提取 id="chatGPTURL" 中的連結

輸出格式

檔案捷徑

  • 按建立日期自動分類
  • 每個檔案生成對應的 .lnk 捷徑
  • 便於快速存取原始檔案

HTML 報告

  • 按日期分組顯示所有搜尋結果
  • 每個項目包含檔案名稱和對應的 AI 對話 URL
  • 支援直接點擊連結開啟對話

自訂設定

修改搜尋路徑

$pathLimit = '您的目標目錄路徑'

調整日期範圍

$dateRange = 'dc:開始日期..結束日期'

變更輸出目錄

$BASE_OUTPUT_FOLDER = "您的輸出目錄名稱"

注意事項

  • 確保 Everything 搜尋引擎正在運行
  • 搜尋路徑必須存在且可存取
  • HTML 檔案必須包含正確的 URL 標記格式
  • 建議定期備份搜尋結果

故障排除

常見問題

  1. 找不到 es.exe

    • 檢查 Everything 安裝路徑
    • 確認 $ES_PATH 變數設定正確

  2. 無法提取 URL

    • 檢查 HTML 檔案格式
    • 確認包含正確的 URL 標記

  3. 搜尋結果為空

    • 檢查搜尋路徑是否存在
    • 確認日期範圍設定
    • 驗證檔案命名格式

版本資訊

  • 當前版本：基於 ES-1.1.0.10
  • 支援平台：Windows PowerShell
  • 最後更新：2025年
