磁碟同步差異工具 (Sync Diff Disk)

這是一個用於比較和同步不同磁碟間資料夾結構的 Python 工具。它可以分析兩個目錄的結構差異，並建議或執行檔案/資料夾的移動操作。

功能特色

  • 🔍 結構比較: 比較兩個目錄的資料夾結構
  • 📁 智能移動: 自動建議需要移動的資料夾
  • 🛡️ 安全操作: 避免覆蓋現有檔案，自動添加編號後綴
  • 📝 詳細日誌: 提供完整的操作日誌記錄
  • ⚡ 深度控制: 可設定掃描深度，避免過度遍歷

系統需求

  • Python 3.6 或更高版本
  • Windows 作業系統（支援 Windows 路徑格式）

安裝與使用

1. 下載專案

git clone <repository-url>
cd sync-diff-disk

2. 執行應用程式

python app.py

3. 配置路徑

在 app.py 檔案中修改以下變數：

source_path = r"G:\youtube"      # 源目錄路徑
target_path = r"K:\youtube"      # 目標目錄路徑

主要功能說明

get_folder_structure(path, depth=4)

獲取指定路徑的資料夾結構，預設掃描深度為 4 層。

參數:

  • path: 要掃描的目錄路徑
  • depth: 掃描深度（預設為 4）

返回:

  • 包含目錄結構的字典

compare_and_suggest_moves(source_path, target_path)

比較兩個目錄結構並建議需要移動的資料夾。

參數:

  • source_path: 源目錄路徑
  • target_path: 目標目錄路徑

返回:

  • 包含移動建議的列表，每個元素為 (舊路徑, 新路徑) 的元組

execute_moves(moves)

執行建議的移動操作。

參數:

  • moves: 移動操作列表

功能:

  • 自動創建目標目錄的父目錄
  • 如果目標路徑已存在，自動添加編號後綴
  • 記錄所有操作和錯誤

使用範例

# 設定源目錄和目標目錄
source_path = r"D:\00_DIY"
target_path = r"I:\00_DIY"

# 獲取移動建議
suggested_moves = compare_and_suggest_moves(source_path, target_path)

# 顯示建議的移動操作
print("建議的移動操作：")
for old_path, new_path in suggested_moves:
    print(f"從 {old_path} 移動到 {new_path}")

# 執行移動操作
execute_moves(suggested_moves)

日誌記錄

應用程式使用 Python 的 logging 模組記錄操作過程：

  • INFO: 正常操作記錄
  • WARNING: 警告訊息（如源路徑不存在）
  • ERROR: 錯誤訊息（如移動操作失敗）

注意事項

⚠️ 重要提醒:

  1. 備份資料: 在執行移動操作前，請務必備份重要資料
  2. 路徑格式: 請使用正確的 Windows 路徑格式（使用 r"..." 或雙反斜線）
  3. 權限檢查: 確保對源目錄和目標目錄有適當的讀寫權限
  4. 磁碟空間: 確保目標磁碟有足夠的空間

故障排除

常見問題

  1. 權限錯誤

    • 確保以管理員身份執行
    • 檢查目錄權限設定

  2. 路徑不存在

    • 檢查路徑是否正確
    • 確保磁碟已正確掛載

  3. 磁碟空間不足

    • 清理目標磁碟空間
    • 分批執行移動操作
