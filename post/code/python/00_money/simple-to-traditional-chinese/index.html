簡體轉繁體中文工具

這是一個用於將簡體中文轉換為繁體中文的工具集，支援多種使用方式，包括剪貼簿轉換和批量文件處理。

功能特色

  • 🔄 剪貼簿轉換：快速轉換剪貼簿中的簡體中文為繁體中文
  • 📁 批量文件處理：自動處理指定目錄下的所有 Markdown 文件
  • 🎯 多種使用方式：支援 Python 腳本、PowerShell 腳本和批次檔
  • 📊 深度控制：可設定目錄遍歷深度，避免處理過深的子目錄
  • 🛡️ 錯誤處理：具備完善的錯誤處理機制

專案結構

app/
├── app.py                    # 主要 Python 腳本（批量處理 Markdown 文件）
├── translate.ps1             # PowerShell 腳本（剪貼簿轉換）
├── goo.bat                   # 批次檔（執行 Python 腳本）
├── issue.markdown            # 問題記錄文件
└── 00_version/               # 版本歷史
    ├── v1_clipboard-ok;wait-wlak-through-centrain folder.py
    └── v2_all-target-folder-md,markdown-ok;wait-current-folder-cmd-bat.py

安裝需求

Python 環境

pip install opencc-python-reimplemented

PowerShell 環境

需要安裝 Microsoft.International.Converters.PinYinConverter 組件（Windows 系統通常已內建）

使用方法

方法一：剪貼簿轉換（PowerShell）

  1. 複製需要轉換的簡體中文文字到剪貼簿
  2. 執行 PowerShell 腳本：
.\translate.ps1
  1. 轉換後的繁體中文會自動存回剪貼簿

方法二：批量文件處理（Python）

  1. 修改 app.py 中的 target_path 變數，設定目標目錄：
target_path = r"你的目標目錄路徑"
  1. 執行 Python 腳本：
python app.py

或使用批次檔：

goo.bat

方法三：命令列參數（可選）

取消註解 app.py 中的命令列參數處理代碼，即可透過命令列指定目標路徑：

python app.py "目標目錄路徑"

  • 支援目錄深度控制
  • 改進錯誤處理機制

當前版本

  • 整合所有功能
  • 支援多種使用方式

技術細節

  • 轉換引擎：使用 OpenCC (Open Chinese Convert) 進行簡繁轉換
  • 文件格式：支援 .md 和 .markdown 文件
  • 編碼：使用 UTF-8 編碼處理文件
  • 深度限制：預設最大遍歷深度為 4 層目錄

注意事項

  1. 請確保目標目錄有寫入權限
  2. 建議在處理重要文件前先備份
  3. 轉換過程會直接覆蓋原始文件
  4. 如遇到編碼問題，請確保文件使用 UTF-8 編碼

故障排除

常見問題

Q: 執行時出現 “ModuleNotFoundError: No module named ‘opencc’” A: 請安裝 opencc 庫：pip install opencc-python-reimplemented

Q: PowerShell 腳本無法執行 A: 請檢查執行政策設定：Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Q: 文件轉換後出現亂碼 A: 請確保原始文件使用 UTF-8 編碼
