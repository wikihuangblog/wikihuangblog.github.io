簡體轉繁體中文轉換器 (Chinese Converter)

一個輕量級的 Windows 桌面應用程式，可以快速將簡體中文轉換為繁體中文。程式會在系統托盤中運行，並提供快捷鍵功能，讓您隨時隨地進行文字轉換。

功能特色

  • 🔄 一鍵轉換：使用快捷鍵 Ctrl+Alt+C 快速轉換剪貼簿中的簡體中文為繁體中文
  • 🖥️ 系統托盤：程式在系統托盤中運行，不佔用桌面空間
  • 🚀 開機自啟：自動設定開機啟動，無需手動開啟
  • 📋 剪貼簿整合：直接讀取和寫入剪貼簿內容，操作簡單直觀
  • 🎯 高精度轉換：使用 OpenCC 引擎，提供準確的簡繁轉換

系統需求

  • Windows 10/11
  • Python 3.7+ (僅開發環境需要)

安裝與使用

方法一：使用預編譯執行檔 (推薦)

  1. 下載 ChineseConverter.exe 執行檔
  2. 雙擊執行即可開始使用
  3. 程式會自動在系統托盤中顯示圖標
  4. 使用 Ctrl+Alt+C 快捷鍵進行文字轉換

方法二：從原始碼編譯

  1. 確保已安裝 Python 3.7 或更高版本
  2. 安裝所需套件： 
    pip install keyboard pyperclip opencc-python-reimplemented pystray pillow pyinstaller
  3. 執行編譯腳本： 
    python setup.py
  4. 編譯完成後，在 build/exe.win-amd64-3.11/ 目錄中找到 ChineseConverter.exe

使用說明

  1. 啟動程式：執行 ChineseConverter.exe

  2. 轉換文字：

    • 複製需要轉換的簡體中文文字到剪貼簿
    • 按下 Ctrl+Alt+C 快捷鍵
    • 轉換後的繁體中文會自動替換剪貼簿中的內容
    • 貼上即可使用轉換後的文字

  3. 退出程式：右鍵點擊系統托盤中的圖標，選擇「退出」

專案結構

app/
├── app.py                 # 主程式檔案
├── setup.py              # PyInstaller 編譯腳本
├── icon.png              # 程式圖標
├── ChineseConverter/     # 編譯後的執行檔目錄
│   ├── app.py
│   ├── setup.bat
│   └── start.bat
└── build/                # 編譯輸出目錄
    └── exe.win-amd64-3.11/
        └── ChineseConverter.exe

技術細節

  • 轉換引擎：OpenCC (Open Chinese Convert)
  • GUI 框架：Pystray (系統托盤)
  • 快捷鍵監聽：keyboard
  • 剪貼簿操作：pyperclip
  • 圖像處理：Pillow (PIL)
  • 編譯工具：PyInstaller

開發說明

依賴套件

keyboard>=1.13.0
pyperclip>=1.8.0
opencc-python-reimplemented>=0.1.6
pystray>=0.19.0
pillow>=8.0.0
pyinstaller>=4.0

編譯指令

# 使用 PyInstaller 編譯為單一執行檔
pyinstaller --onefile --windowed --icon=icon.png --name=ChineseConverter --add-data=icon.png;. app.py

注意事項

  • 程式會自動設定開機啟動，如需取消可在 Windows 啟動項中移除
  • 首次執行可能需要管理員權限來設定開機啟動
  • 快捷鍵 Ctrl+Alt+C 可能與其他程式衝突，請注意避免
