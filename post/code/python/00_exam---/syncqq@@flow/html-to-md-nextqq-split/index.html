HTML 轉 Markdown 工具

這是一個專門用於將 HTML 內容轉換為 Markdown 格式的 Python 工具，特別針對豆包（Doubao）聊天記錄進行優化處理。

功能特色

  • HTML 到 Markdown 轉換：使用 BeautifulSoup 和 html2text 庫進行高品質轉換
  • 智能內容處理：自動識別並處理「下一題」等關鍵詞，將其轉換為標題格式
  • 格式清理：移除不必要的 HTML 標籤和樣式，保持內容簡潔
  • 中文支援：完整支援繁體中文和簡體中文內容
  • 代碼塊處理：特別優化代碼標籤的處理，確保代碼格式正確

專案結構

app/
├── app.py                          # 主程式文件
├── input.html                      # 輸入的 HTML 文件
├── output.md                       # 輸出的 Markdown 文件
├── README.md                       # 專案說明文件
├── 00_different-input/             # 不同輸入範例
│   ├── decimal-to-multi-base/      # 十進制轉多進制範例
│   └── 程序算法小考官---.doubao.html
├── 00_version/                     # 版本歷史
│   ├── v1.py                       # 第一版（空文件）
│   ├── v2_newQ-translate.py        # 第二版（空文件）
│   └── v3_newQ-as-h3-ok.py        # 第三版（完整功能）
└── split_sections/                 # 分割後的章節文件

安裝需求

確保您的系統已安裝以下 Python 套件：

pip install beautifulsoup4 html2text

使用方法

基本使用

  1. 將要轉換的 HTML 文件命名為 input.html 並放在專案根目錄
  2. 執行主程式：
python app.py
  1. 轉換完成後，結果將保存在 output.md 文件中

自定義輸入輸出文件

您可以修改 app.py 中的文件路徑：

if __name__ == "__main__":
    input_file = "your_input.html"    # 修改輸入文件名
    output_file = "your_output.md"    # 修改輸出文件名
    main(input_file, output_file)
    print(f"轉換完成！結果已保存到 {output_file}")

核心功能說明

1. HTML 轉換功能 (convert_html_to_md)

  • 使用 BeautifulSoup 解析 HTML 內容
  • 移除 script 和 style 標籤
  • 特別處理 code 標籤，用三個反引號包裹
  • 使用 html2text 進行 Markdown 轉換

2. 智能標題識別 (is_next_question)

自動識別包含以下關鍵詞的內容：

  • 下一題（繁體）
  • 下一题（簡體）
  • 下一道題
  • 下一道题
  • 下一个题目
  • 下一個題目
  • 下一个新的问题

3. 格式清理 (clean_markdown_formatting)

清除以下 Markdown 格式：

  • 標題格式（# 開頭）
  • 粗體和斜體標記
  • 行內代碼和代碼塊標記
  • 列表標記
  • 引用標記

4. 內容處理 (process_markdown_content)

  • 逐行處理 Markdown 內容
  • 將包含「下一題」的行轉換為三級標題（###）
  • 保持其他內容不變

技術細節

依賴套件

  • beautifulsoup4：HTML 解析和處理
  • html2text：HTML 到 Markdown 轉換
  • re：正則表達式處理（Python 內建）

編碼支援

  • 支援 UTF-8 編碼
  • 完整支援中文字符
  • 保持 Unicode 字符完整性

注意事項

  1. 確保輸入的 HTML 文件使用 UTF-8 編碼
  2. 程式會自動處理各種 HTML 標籤，但建議輸入文件結構清晰
  3. 轉換後的 Markdown 文件會覆蓋同名的現有文件
  4. 程式特別針對豆包聊天記錄進行優化，對其他 HTML 內容也有良好支援

故障排除

常見問題

  1. 編碼錯誤：確保 HTML 文件使用 UTF-8 編碼
  2. 套件缺失：執行 pip install beautifulsoup4 html2text 安裝依賴
  3. 文件路徑錯誤：檢查輸入文件是否存在於指定路徑

錯誤處理

程式包含基本的錯誤處理機制，如果遇到問題，請檢查：

  • 輸入文件是否存在
  • 文件編碼是否正確
  • 依賴套件是否已安裝
