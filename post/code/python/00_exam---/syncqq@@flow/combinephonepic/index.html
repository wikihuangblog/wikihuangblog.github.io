手機截圖合併工具

這是一個用於自動合併手機截圖的 Python 工具，可以將多張手機截圖裁切並水平合併成一張圖片。

功能特色

  • 自動識別並處理以 Screenshot_ 開頭的 JPG 檔案
  • 自動裁切截圖的上下部分（上方裁切 350 像素，下方裁切 500 像素）
  • 將多張截圖水平合併成一張圖片
  • 與 QQ.jpg 檔案進行最終合併
  • 處理完成後自動清理原始截圖檔案

系統需求

  • Python 3.x
  • PIL (Pillow) 套件

安裝依賴

pip install Pillow

使用方法

方法一：使用批次檔（推薦）

  1. 將所有需要合併的截圖檔案（以 Screenshot_ 開頭）和 QQ.jpg 放在同一個資料夾中
  2. 雙擊執行 00_phone-pic-combine.bat
  3. 程式會自動處理並生成 00_merge.jpg

方法二：直接執行 Python 腳本

python app.py "資料夾路徑"

例如：

python app.py "C:\Users\YourName\Pictures\Screenshots"

檔案結構

app/
├── app.py                      # 主要程式檔案
├── 00_phone-pic-combine.bat    # 批次執行檔
├── README.md                   # 說明文檔
├── issue.txt                   # 開發記錄
└── backupppp/                  # 備份資料夾
    ├── 00_merge.jpg
    ├── QQ.jpg
    └── Screenshot_*.jpg

處理流程

  1. 檔案識別：掃描指定資料夾中所有以 Screenshot_ 開頭的 JPG 檔案
  2. 排序處理：按檔案名稱升序排列
  3. 圖片裁切：每張截圖裁切上方 350 像素和下方 500 像素
  4. 第一次合併：將所有裁切後的截圖水平合併成 00_merge.jpg
  5. 最終合併：將 QQ.jpg 與 00_merge.jpg 合併
  6. 清理檔案：刪除所有原始 Screenshot_ 開頭的檔案

輸出結果

  • 生成檔案：00_merge.jpg（包含 QQ.jpg 和所有截圖的合併結果）
  • 自動清理：刪除所有原始截圖檔案

注意事項

  • 確保資料夾中包含 QQ.jpg 檔案
  • 截圖檔案必須以 Screenshot_ 開頭
  • 程式會自動覆蓋已存在的 00_merge.jpg 檔案
  • 建議在執行前備份重要檔案
