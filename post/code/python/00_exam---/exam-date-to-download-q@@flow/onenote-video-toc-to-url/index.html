OneNote 影片目錄轉 URL 工具

這是一個 Python 工具，用於將 OneNote 影片目錄（Markdown 格式）轉換為資料夾結構和 URL 檔案，方便下載和管理影片資源。

功能特色

  • 📁 自動創建資料夾結構
  • 🔗 生成 .url 檔案，快速訪問影片連結
  • 🧹 自動清理不合法的資料夾名稱
  • 📝 支援 Markdown 格式的目錄檔案
  • 🎯 專門針對考試影片資源優化

系統需求

  • Python 3.6 或更高版本
  • Windows 作業系統（支援 .url 檔案）

安裝與使用

方法一：使用批次檔（推薦）

  1. 將您的 Markdown 目錄檔案放在專案目錄中
  2. 執行批次檔： 
    go.bat your_toc_file.md

方法二：直接使用 Python

  1. 開啟命令提示字元或 PowerShell
  2. 執行以下命令： 
    python app.py your_toc_file.md

支援的目錄格式

工具支援以下 Markdown 格式：

有 URL 連結的格式

- [113年度](https://example.com/video1)
- [112年度](https://example.com/video2)

純文字格式

- 113年度影片
- 112年度影片

行號格式（自動處理）

1||- [113年度](https://example.com/video1)
2||- [112年度](https://example.com/video2)

輸出結果

執行後會在 output 資料夾中生成：

output/
├── 113年度/
│   └── 113年度.url
├── 112年度/
│   └── 112年度.url
└── 111年度/
    └── 111年度.url

內建範例檔案

專案包含以下範例目錄檔案：

  • toc_postoffice.md - 郵局考試影片目錄
  • toc_twport.md - 台灣港務公司考試影片目錄
  • toc-mrt.md - 台北捷運考試影片目錄
  • memo.md - 備忘錄和筆記

使用範例

處理郵局考試影片目錄

python app.py toc_postoffice.md

處理港務公司考試影片目錄

python app.py toc_twport.md

處理捷運考試影片目錄

python app.py toc-mrt.md

資料夾名稱清理規則

工具會自動清理資料夾名稱，移除以下不合法字元：

  • 開頭和結尾的特殊字元：| -[]().
  • Windows 不支援的字元：<>:"/\|?*
  • 多餘的空格和破折號

錯誤處理

  • 如果資料夾名稱清理後為空，會使用預設名稱 unnamed_folder
  • 如果無法創建資料夾，會顯示錯誤訊息但繼續處理其他項目
  • 如果指定的 Markdown 檔案不存在，會顯示錯誤並退出

注意事項

  • 確保有足夠的磁碟空間創建資料夾和檔案
  • 建議在執行前備份重要的資料
  • 生成的 .url 檔案可以在 Windows 檔案總管中直接點擊開啟

授權

此專案為個人使用工具，請根據需要自行修改和分發。
