HTML 文件分割工具

這是一個用於將大型 HTML 文件分割成多個小文件的 Python 工具，特別適用於處理包含題目和章節的 HTML 文件。

功能特色

  • 🔧 自動識別 HTML 文件中的標題結構（H1 和 H3 標題）
  • 📝 智能文件名清理和格式化
  • 🎯 支援題目編號提取和格式化
  • 📁 自動創建輸出目錄
  • 🌐 支援中文文件名和內容

專案結構

app/
├── app.py                                    # 主程式文件
├── book-chapter.html                        # 輸入的 HTML 文件
├── memo.markdown                            # 專案備忘錄
├── output/                                  # 輸出目錄
│   ├── 01.html
│   ├── 02.html
│   └── ...
├── 112-data-info-book-chapter-backup/       # 備份目錄
│   ├── 112-data-info-book-chapter.html
│   └── output/                              # 備份輸出
└── README.md                                # 本文件

安裝需求

Python 版本

  • Python 3.6 或更高版本

依賴套件

pip install beautifulsoup4

使用方法

1. 準備輸入文件

將要分割的 HTML 文件命名為 book-chapter.html 並放置在專案根目錄。

2. 執行程式

python app.py

3. 查看結果

分割後的 HTML 文件將保存在 output/ 目錄中，每個文件對應一個章節或題目。

程式功能說明

主要函數

clean_filename(filename)

  • 清理文件名，移除特殊字符
  • 保留字母、數字和中文字符
  • 將多個連續的 ‘-’ 替換為單個 ‘-’

extract_question_number(filename)

  • 提取題目編號
  • 支援 “題目-數字” 和 “數字-題目” 兩種格式

format_number(number)

  • 格式化數字，為 1-9 的數字添加前導零

extract_number_prefix(title)

  • 提取標題中的數字前綴
  • 標準化為 “x-x-0” 格式

split_and_save_html(html_content)

  • 主要分割函數
  • 識別 H1 和 H3 標題
  • 將每個章節保存為獨立的 HTML 文件

輸出格式

每個分割後的 HTML 文件包含：

  • 完整的 HTML 結構
  • UTF-8 編碼
  • 對應的標題
  • 該章節的所有內容

範例

輸入

<h1 class="header-vfC6AV">9-1 計算機基礎</h1>
<p>計算機基礎知識內容...</p>
<h3 class="header-vfC6AV">題目 1</h3>
<p>題目內容...</p>

輸出

  • 9-1-0-計算機基礎.html
  • 01.html

注意事項

  1. 確保輸入的 HTML 文件使用 UTF-8 編碼
  2. 程式會自動創建 output 目錄
  3. 如果輸出目錄已存在，會覆蓋同名文件
  4. 支援中文文件名，但建議在支援中文的系統上運行

故障排除

常見問題

  1. 編碼錯誤

    • 確保 HTML 文件使用 UTF-8 編碼
    • 檢查系統是否支援中文文件名

  2. 找不到標題

    • 確認 HTML 文件中的標題使用正確的 CSS 類別 header-vfC6AV
    • 檢查標題是否為 H1 或 H3 標籤

  3. 文件權限問題

    • 確保程式有權限在當前目錄創建文件和文件夾
