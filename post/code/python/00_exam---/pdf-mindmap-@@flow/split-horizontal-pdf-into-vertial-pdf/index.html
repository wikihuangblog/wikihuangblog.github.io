PDF 水平分割工具

這是一個用於將水平 PDF 文件分割成垂直 PDF 文件的 Python 工具。該工具可以將每個 PDF 頁面從中間水平分割，將原本的橫向頁面轉換為兩個縱向頁面。

功能特色

  • 🔄 自動將水平 PDF 分割成垂直 PDF
  • 📁 批量處理目標文件夾中的所有 PDF 文件
  • 🛡️ 錯誤處理機制，確保程序穩定性
  • 📦 使用 PyMuPDF (fitz) 庫，提供高質量的 PDF 處理

系統需求

  • Python 3.6 或更高版本
  • PyMuPDF 庫

安裝依賴

pip install PyMuPDF

使用方法

方法一：使用批處理文件（推薦）

  1. 將要處理的 PDF 文件放入 target 文件夾
  2. 雙擊運行 run.bat 文件
  3. 處理完成的文件將保存在 target/split_pdfs/ 文件夾中

方法二：直接運行 Python 腳本

python app.py [目標文件夾路徑]

例如：

python app.py E:\codeAll\python\00_exam---\pdf-mindmap-@@flow\split-horizontal-pdf-into-vertial-pdf\app

文件結構

app/
├── app.py                    # 主程序文件
├── run.bat                   # Windows 批處理執行文件
├── issue.txt                 # 專案說明
├── target/                   # 目標 PDF 文件存放文件夾
│   ├── *.pdf                # 待處理的 PDF 文件
│   └── split_pdfs/          # 處理後的 PDF 文件輸出文件夾
└── 00_version/              # 版本歷史文件夾
    ├── v1_chinese-pdf-fail;try-print-no-text-version.py  # v1 版本（使用 PyPDF2）
    └── v2_PyMuPDF.py        # v2 版本（使用 PyMuPDF）

工作原理

  1. 讀取 PDF：使用 PyMuPDF 打開目標 PDF 文件
  2. 頁面分割：對每一頁進行水平分割
    • 計算頁面中心點位置
    • 創建左半部分和右半部分的矩形區域
    • 將每個部分作為新頁面添加到輸出 PDF
  3. 保存結果：將處理後的 PDF 保存到指定文件夾

版本歷史

v1 版本

  • 使用 PyPDF2 庫
  • 在處理中文 PDF 時遇到問題
  • 嘗試使用無文字版本但效果不佳

v2 版本（當前版本）

  • 改用 PyMuPDF (fitz) 庫
  • 更好的 PDF 兼容性
  • 成功處理各種類型的 PDF 文件

注意事項

  • 確保目標文件夾中有 PDF 文件
  • 處理大型 PDF 文件時可能需要較長時間
  • 輸出文件會自動添加 split_ 前綴
  • 如果處理過程中出現錯誤，程序會繼續處理其他文件

錯誤處理

程序包含完整的錯誤處理機制：

  • 如果某個 PDF 文件無法處理，會顯示錯誤信息並繼續處理下一個文件
  • 所有錯誤信息都會在控制台中顯示

技術細節

  • 主要庫：PyMuPDF (fitz)
  • 處理方式：頁面級別的分割
  • 輸出格式：保持原始 PDF 質量
  • 文件命名：自動添加 split_ 前綴

授權

此專案為開源專案，可自由使用和修改。
