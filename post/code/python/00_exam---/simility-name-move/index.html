Simility Name Move

一個基於文件名相似度的智能文件移動工具，用於自動整理和組織 HTML 文件。

功能概述

這個項目包含兩個主要的 Python 腳本，用於根據文件名相似度自動移動 HTML 文件到相應的目標文件夾：

  1. app.py - 基於文件夾名稱匹配的移動工具
  2. simility-move.py - 基於 Levenshtein 距離算法的智能相似度匹配工具

文件說明

app.py

  • 功能: 將 HTML 文件移動到名稱以 _{basename} 結尾的目標文件夾
  • 特點: 簡單直接的匹配方式，適合有明確命名規則的文件夾結構
  • 使用場景: 當目標文件夾有固定的命名格式時

simility-move.py

  • 功能: 使用 Levenshtein 距離算法計算文件名與文件夾名的相似度
  • 特點: 智能匹配，相似度閾值為 0.92
  • 使用場景: 當需要更靈活的匹配，或文件名與文件夾名不完全一致時

安裝依賴

pip install python-Levenshtein

使用方法

方法一：使用 app.py

  1. 修改腳本中的路徑變量：

    ori_html_folder = r'你的原始HTML文件夾路徑'
target_base_path = r'你的目標基礎路徑'

  2. 運行腳本：

    python app.py

方法二：使用 simility-move.py

  1. 修改腳本中的路徑變量：

    target_folder = r'你的目標文件夾路徑'
ori_html_folder = r'你的原始HTML文件夾路徑'

  2. 運行腳本：

    python simility-move.py

核心功能

字符串處理

  • 自動移除文件名中的數字
  • 去除前導空格
  • 提取文件基本名稱

相似度計算

  • 使用 Levenshtein 距離算法
  • 可調整的相似度閾值
  • 智能匹配最佳目標文件夾

文件操作

  • 安全的文件移動操作
  • 重複文件檢測
  • 錯誤處理和日誌記錄

注意事項

  1. 備份重要文件: 在運行腳本前，建議備份重要的 HTML 文件
  2. 路徑設置: 確保所有路徑變量設置正確
  3. 權限檢查: 確保腳本有足夠的權限讀取和移動文件
  4. 相似度閾值: 可以根據需要調整 max_similarity 值

錯誤處理

腳本包含完善的錯誤處理機制：

  • 文件已存在時會跳過移動
  • 移動失敗時會顯示錯誤信息
  • 找不到匹配文件夾時會給出提示

適用場景

  • 整理大量 HTML 文件
  • 自動化文件歸類
  • 批量文件移動
  • 基於名稱相似度的文件組織

技術特點

  • 使用 Python 標準庫 os 和 shutil
  • 集成 Levenshtein 距離算法
  • 支持遞歸目錄遍歷
  • 提供詳細的操作日誌

版本信息

  • Python 版本要求: 3.6+
  • 主要依賴: python-Levenshtein
  • 開發環境: Windows 10
