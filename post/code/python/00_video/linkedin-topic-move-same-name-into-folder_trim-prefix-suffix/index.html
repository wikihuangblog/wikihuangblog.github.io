LinkedIn 主題檔案整理工具

這是一個用於自動整理 LinkedIn 相關檔案的 Python 工具，可以根據檔案名稱的前綴和後綴來分組並移動檔案到對應的資料夾中。

功能特色

  • 🔄 自動識別檔案名稱的前綴（數字_）和後綴（_@@、.lnk、.jpg、.zip 等）
  • 📁 根據核心檔案名稱自動分組檔案
  • 🚀 將相同主題的檔案移動到對應的資料夾中
  • 🛡️ 支援多種檔案格式（.jpg、.lnk、.zip、.html、.srt）
  • ⚡ 提供批次執行腳本

檔案說明

主要腳本

  1. linkedin-topic-move-same-name-into-folder_trim-prefix-suffix.py

    • 完整版本，包含自動創建資料夾功能
    • 支援更多檔案格式（包含 .zip）

  2. linkedin-topic-move-same-name-into-folder_trim-prefix-suffix-no-auto-create.py

    • 簡化版本，不自動創建資料夾
    • 支援基本檔案格式（.jpg、.lnk）

輔助檔案

  • cmd/run.bat - Windows 批次執行腳本
  • regex101.url - 正則表達式測試工具連結

使用方法

方法一：直接執行 Python 腳本

python linkedin-topic-move-same-name-into-folder_trim-prefix-suffix.py "目標資料夾路徑"

方法二：使用批次腳本（Windows）

  1. 將 run.bat 複製到您要整理的資料夾中
  2. 雙擊執行 run.bat

檔案命名規則

工具會自動識別以下格式的檔案：

  • 前綴格式：數字_ （例如：001_、123_）
  • 後綴格式：
    • _@@
    • .lnk
    • .jpg
    • .zip
    • .html
    • .srt

範例

假設您有以下檔案：

001_LinkedIn_Marketing_@@.jpg
002_LinkedIn_Marketing_@@.lnk
003_LinkedIn_Marketing_@@.zip
001_LinkedIn_Networking_@@.jpg
002_LinkedIn_Networking_@@.lnk

執行後會創建以下資料夾結構：

LinkedIn_Marketing/
├── 001_LinkedIn_Marketing_@@.jpg
├── 002_LinkedIn_Marketing_@@.lnk
└── 003_LinkedIn_Marketing_@@.zip

LinkedIn_Networking/
├── 001_LinkedIn_Networking_@@.jpg
└── 002_LinkedIn_Networking_@@.lnk

系統需求

  • Python 3.6 或更高版本
  • Windows 系統（批次腳本）

注意事項

⚠️ 重要提醒：

  • 執行前請確保已備份重要檔案
  • 工具會移動檔案，請確認目標資料夾路徑正確
  • 建議先在測試資料夾中試用

錯誤處理

工具包含錯誤處理機制，如果移動檔案時發生錯誤，會顯示錯誤訊息但不會中斷整個程序。

自定義設定

您可以修改腳本中的以下設定：

  • 檔案格式：修改 allowed_extensions 變數
  • 正則表達式：調整 get_core_filename 函數中的匹配規則
  • 資料夾創建：選擇是否自動創建資料夾（註解/取消註解相關程式碼）

此專案為開源軟體，可自由使用和修改。
