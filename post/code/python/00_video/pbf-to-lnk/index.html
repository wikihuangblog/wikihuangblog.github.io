PBF 到 LNK 轉換器

這是一個Python應用程序，用於處理視頻時間文件（.pbf格式）並自動創建視頻播放器的快捷方式，讓您可以快速跳轉到視頻的特定時間點。

功能特點

  • 解析 .pbf 格式的視頻時間文件
  • 自動創建視頻播放器快捷方式（.lnk文件）
  • 支持自定義視頻播放器路徑
  • 自動在輸入文件同目錄創建 output 文件夾
  • 支持命令行參數和批次檔執行

項目結構

app/
├── assets/                    # 示例 .pbf 文件
│   ├── migii-toeic-exam-1.pbf
│   ├── videoTime-backup.pbf
│   └── videoTime.pbf
├── lib/                       # 核心功能模組
│   ├── shortcut_creator.py    # 快捷方式創建器
│   └── video_processor.py     # 視頻時間文件處理器
├── little/                    # 輔助工具
│   ├── little.py
│   ├── timeTolnk.py
│   └── video-time-pic.py
├── main.py                    # 主程序（固定路徑版本）
├── main2.py                   # 主程序（命令行參數版本）
├── test_video_processor.py    # 測試文件
└── gooo.bat                   # 批次執行檔

安裝要求

Python 依賴

  • Python 3.6+
  • pywin32 - 用於創建Windows快捷方式

安裝依賴

pip install pywin32

使用方法

方法一：使用命令行參數（推薦）

python main2.py "路徑/到/你的/文件.pbf" --video_exe_path "C:\Program Files\DAUM\PotPlayer\PotPlayerMini64.exe"

方法二：使用批次檔

  1. 編輯 gooo.bat 文件，修改其中的文件路徑
  2. 雙擊執行批次檔

方法三：修改 main.py

  1. 編輯 main.py 文件
  2. 修改 input_file 變數為你的 .pbf 文件路徑
  3. 修改 exe_path 變數為你的視頻播放器路徑
  4. 執行：python main.py

輸入文件格式

.pbf 文件格式示例：

[Bookmark]
0=34628*1_認識記憶*[二進制數據]
1=37631*_1.1_記憶的概念與特性_12*[二進制數據]
2=47921*_1.2_記憶法的發展_18*[二進制數據]

格式說明：

  • 第一行：[Bookmark] 標題
  • 後續行：索引=時間毫秒*標題*[二進制數據]
  • 時間會自動從毫秒轉換為秒

輸出結果

程序會在輸入文件同目錄下創建 output 文件夾，並在其中生成對應的快捷方式文件：

  • 1_認識記憶.lnk - 跳轉到 34.628 秒
  • _1.1_記憶的概念與特性_12.lnk - 跳轉到 37.631 秒
  • _1.2_記憶法的發展_18.lnk - 跳轉到 47.921 秒

支持的視頻播放器

  • PotPlayer（默認）：C:\Program Files\DAUM\PotPlayer\PotPlayerMini64.exe
  • VLC Media Player：C:\Program Files\VideoLAN\VLC\vlc.exe
  • Windows Media Player：C:\Program Files\Windows Media Player\wmplayer.exe
  • 其他支持命令行參數的視頻播放器

命令行參數

main2.py 參數

  • input_file（必需）：.pbf 文件的完整路徑
  • --video_exe_path（可選）：視頻播放器執行檔路徑，默認為 PotPlayer

示例

# 使用默認 PotPlayer
python main2.py "D:\videos\my_video.pbf"

# 使用 VLC
python main2.py "D:\videos\my_video.pbf" --video_exe_path "C:\Program Files\VideoLAN\VLC\vlc.exe"

測試

運行測試文件以驗證功能：

python test_video_processor.py

注意事項

  1. 確保 .pbf 文件與對應的 .mp4 視頻文件在同一目錄下
  2. 視頻播放器必須支持命令行參數（如 /seek= 或 --start-time=）
  3. 程序會自動創建 output 文件夾，如果已存在則會覆蓋其中的快捷方式
  4. 快捷方式文件名會使用 .pbf 文件中的標題，特殊字符會被保留

故障排除

常見問題

  1. 找不到視頻文件

    • 確保 .mp4 文件與 .pbf 文件在同一目錄
    • 檢查文件名是否匹配（除了擴展名）

  2. 快捷方式無法播放

    • 檢查視頻播放器路徑是否正確
    • 確認播放器支持命令行參數

  3. 編碼錯誤

    • 確保 .pbf 文件使用 UTF-8 編碼
    • 檢查文件是否包含特殊字符

開發者信息

  • 視頻內容的快速導航
