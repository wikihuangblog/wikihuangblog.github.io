視頻文件重命名工具

這是一個用於自動重命名視頻文件和字幕文件的 Python 工具，特別適用於處理 Lynda 課程文件。

功能特點

  • 自動重命名 MP4 文件：將複雜的文件名格式轉換為簡潔的格式
  • 自動重命名 SRT 字幕文件：提取並標準化字幕文件名
  • 統一命名規則：確保視頻文件和字幕文件使用相同的命名格式
  • 零填充：自動為數字前綴添加零填充（例如：1.1 → 01.01）

文件結構

app/
├── app.py          # 主程序文件
├── run.bat         # Windows 批處理執行文件
└── README.md       # 說明文檔

使用方法

方法一：使用批處理文件（推薦）

  1. 將 run.bat 和 app.py 複製到包含需要重命名的視頻文件的目錄
  2. 雙擊運行 run.bat

方法二：使用命令行

python app.py <目標文件夾路徑>

例如：

python app.py "C:\Users\YourName\Videos\Course"

重命名規則

MP4 文件重命名

  • 原始格式：數字_0數字.0數字_描述_標題.mp4
  • 新格式：數字.數字_標題.mp4
  • 零填充：數字會自動添加前導零（例如：1.1 → 01.01）

SRT 字幕文件重命名

  • 原始格式：任意前綴_數字.數字_描述.srt
  • 新格式：數字.數字_描述.srt
  • 零填充：數字會自動添加前導零

統一命名

  • 程序會確保 MP4 文件和對應的 SRT 文件使用相同的文件名（除了擴展名）

示例

重命名前：

123_01.02_Introduction_to_Python_Basics.mp4
456_01.02_Introduction_to_Python_Basics.srt
789_02.01_Variables_and_Data_Types.mp4
101_02.01_Variables_and_Data_Types.srt

重命名後：

01.02_Introduction_to_Python_Basics.mp4
01.02_Introduction_to_Python_Basics.srt
02.01_Variables_and_Data_Types.mp4
02.01_Variables_and_Data_Types.srt

系統要求

  • Python 3.x
  • Windows 操作系統（批處理文件）

注意事項

  • 請在運行程序前備份您的文件
  • 程序會直接修改文件名，請確保在正確的目錄中運行
  • 支持的文件格式：.mp4、.srt、.html

錯誤處理

如果遇到以下錯誤：

  • Usage: python app.py <target_folder>：請確保提供了正確的文件夾路徑參數
  • 文件重命名失敗：請檢查文件是否被其他程序佔用，或是否有足夠的權限

開發者信息
