LNK 目標資料夾 Markdown 檔案查找工具

這是一個用於處理 Windows 快捷方式（.lnk）檔案的 Python 工具，能夠自動查找快捷方式指向的資料夾中的 Markdown 檔案，並更新快捷方式的目標路徑。

功能說明

此工具的主要功能是：

  1. 讀取快捷方式列表：從文字檔案中讀取 .lnk 檔案路徑列表
  2. 解析快捷方式目標：獲取每個 .lnk 檔案指向的目標資料夾路徑
  3. 查找 Markdown 檔案：在目標資料夾中搜尋 .md 或 .markdown 檔案
  4. 更新快捷方式：
    • 如果找到 Markdown 檔案，將快捷方式目標更新為該 Markdown 檔案
    • 如果未找到 Markdown 檔案，將原始檔案重命名為 ___no-md-find.lnk

檔案結構

app/
├── app.py                 # 主要程式檔案
├── path-link.txt         # 包含 .lnk 檔案路徑的輸入檔案
├── blog-lnk.txt          # 另一個 .lnk 檔案路徑列表
├── issue.markdown        # 專案需求說明文件
├── all2024.txt           # 2024年相關檔案列表
├── path-link-ori.txt     # 原始路徑檔案備份
├── blog-lnk-ori.txt      # 原始部落格連結檔案備份
└── README.md             # 本說明文件

系統需求

  • Python 3.x
  • Windows 作業系統
  • pywin32 套件（用於處理 Windows 快捷方式）

安裝依賴

pip install pywin32

使用方法

  1. 準備輸入檔案：

    • 將要處理的 .lnk 檔案路徑列表放入 path-link.txt 檔案中
    • 每行一個路徑，確保路徑以 .lnk 結尾

  2. 執行程式：

    python app.py

  3. 查看結果：

    • 程式會輸出處理結果，包括成功更新的快捷方式和未找到 Markdown 檔案的快捷方式
    • 未找到 Markdown 檔案的快捷方式會被重命名為 ___no-md-find.lnk

程式功能詳解

主要函數

  • get_shortcut_target(shortcut_path)：獲取 .lnk 檔案的目標路徑
  • find_markdown_files(folder_path)：在指定資料夾中查找 .md 或 .markdown 檔案
  • create_shortcut(target_path, shortcut_path)：創建新的 .lnk 檔案
  • process_lnk_from_file(lnk_file_path)：從文字檔案讀取並處理 .lnk 路徑

處理流程

  1. 讀取 path-link.txt 檔案中的所有 .lnk 路徑
  2. 對每個路徑：
    • 獲取快捷方式指向的目標資料夾
    • 驗證目標是否為資料夾
    • 在資料夾中搜尋 Markdown 檔案
    • 根據搜尋結果更新或重命名快捷方式

範例

假設有一個快捷方式指向資料夾：

D:\diary\2024\12\00_projproj\01-05-01_112-doubao-rename-prefix-number\

如果該資料夾中包含 memo.markdown 檔案，工具會：

  • 將快捷方式目標更新為 D:\diary\2024\12\00_projproj\01-05-01_112-doubao-rename-prefix-number\memo.markdown

如果該資料夾中沒有 Markdown 檔案，工具會：

  • 將原始快捷方式重命名為 原檔名___no-md-find.lnk

注意事項

  • 此工具僅適用於 Windows 系統
  • 確保有足夠的權限讀取和修改 .lnk 檔案
  • 建議在執行前備份重要的快捷方式檔案
  • 程式會自動跳過無效的路徑和檔案

錯誤處理

程式包含完整的錯誤處理機制：

  • 跳過無效的檔案路徑
  • 處理無法讀取的快捷方式
  • 記錄處理過程中的錯誤訊息

授權

此專案為個人開發工具，僅供學習和研究使用。
