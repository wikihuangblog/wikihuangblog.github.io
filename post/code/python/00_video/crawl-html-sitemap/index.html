Obsidian 文章連結爬蟲工具

這是一個用於處理 Obsidian 相關文章連結的 Python 工具，可以從 sitemap.html 中提取文章連結並自動創建 .url 文件，按照文章編號進行分類整理。

功能特色

  • 🔍 從 sitemap.html 中提取文章標題和連結
  • 📁 根據文章編號自動創建分類資料夾
  • 🏷️ 智能識別多種文章編號格式
  • 📝 自動生成 .url 文件，方便快速訪問
  • 🛡️ 自動處理無效的檔案名稱字符

支援的文章編號格式

工具可以識別以下多種編號格式：

  • Obs171｜文章標題 - 標準格式
  • Obs＃59 文章標題 - 井號格式
  • [Obs＃59] 文章標題 - 方括號格式
  • [Obs#17] 文章標題 - 簡化方括號格式

安裝需求

pip install beautifulsoup4

使用方法

1. 準備 sitemap.html 文件

確保在專案目錄中有 sitemap.html 文件，該文件應包含 Obsidian 相關文章的連結。

2. 執行腳本

python app.py

3. 查看結果

腳本執行後會：

  • 根據文章編號創建對應的資料夾（如：171/, 170/, 169/ 等）
  • 在每個資料夾中創建對應的 .url 文件
  • 無法識別編號的文章會放入 unknown/ 資料夾

輸出範例

假設有以下文章：

Obs171｜WonderBox：美化筆記的外掛，重點提示與強調關鍵
Obs170｜Obsidian的官方網頁擷取工具：Obsidian Web Clipper

執行後會創建：

171/
  └── 171_WonderBox：美化筆記的外掛，重點提示與強調關鍵.url
170/
  └── 170_Obsidian的官方網頁擷取工具：Obsidian Web Clipper.url

檔案結構

app/
├── app.py              # 主要處理腳本
├── sitemap.html        # 輸入的網站地圖文件
├── issue.md           # 專案說明文件
└── README.md          # 本說明文件

程式碼說明

主要函數

  • sanitize_filename() - 清理檔案名稱中的無效字符
  • extract_number() - 從標題中提取文章編號
  • create_folder_by_prefix() - 根據編號創建資料夾
  • process_sitemap() - 主要處理邏輯

檔案名稱處理

  • 自動移除標題中的前綴部分（如 Obs171｜）
  • 將無效字符替換為 -
  • 格式：編號_清理後的標題.url

注意事項

  1. 確保 sitemap.html 文件存在且格式正確
  2. 腳本會自動創建必要的資料夾
  3. 重複執行會覆蓋已存在的 .url 文件
  4. 無法識別編號的文章會統一放入 unknown/ 資料夾

技術細節

  • 使用 BeautifulSoup 解析 HTML
  • 支援多種正則表達式模式匹配
  • 自動處理檔案系統路徑
  • 生成標準的 Windows .url 文件格式
