華藝電子書分類爬蟲工具

專案簡介

這是一個用於解析華藝電子書網站分類結構並自動生成對應 URL 檔案的 Python 工具。該工具可以從 HTML 檔案中提取書籍分類資訊，並為每個分類創建對應的 .url 檔案，方便使用者快速訪問特定分類的書籍頁面。

功能特色

  • 🔍 自動解析 HTML 結構：從華藝電子書網站的 HTML 檔案中提取分類資訊
  • 📁 智能檔案管理：自動創建對應的資料夾結構
  • 🔗 URL 檔案生成：為每個分類生成可直接點擊的 .url 檔案
  • ⚙️ 靈活配置：支援多種處理選項，可自訂是否包含子分類
  • 🛡️ 檔案名稱清理：自動處理不合法的檔案名稱字元

安裝需求

Python 套件依賴

pip install beautifulsoup4

系統需求

  • Python 3.6 或更高版本
  • Windows 系統（用於生成 .url 檔案）

使用方法

1. 基本使用

python app.py

2. 程式配置選項

在 app.py 的 main() 函數中，您可以調整以下參數：

# 設定基礎資料夾名稱
base_folder = "華藝電子書分類"

# 設定是否要排除與主分類相同名稱的子分類
exclude_same_subcats = True  # True 或 False

# 設定是否要包含子分類
include_subcategories = False  # True 或 False

3. 參數說明

  • base_folder：生成的分類資料夾的根目錄名稱
  • exclude_same_subcats：是否排除與主分類名稱相同的子分類
  • include_subcategories：是否處理子分類（設為 False 時只處理主分類）

核心功能說明

1. HTML 解析

程式使用 BeautifulSoup 解析 HTML 檔案，尋找以下元素：

  • 主分類：div.bookSort_first_line > div.bookSort_t
  • 子分類：div.bookSort_content_child_t

2. URL 生成

每個分類會生成對應的 URL：

https://www.airitibooks.com/Search/Category?GroupID={分類ID}&PublishTypeID=0

3. 檔案名稱處理

  • 自動清理不合法的檔案名稱字元（<>:"/\|?*）
  • 將非法字元替換為底線（_）

4. 資料夾結構

程式會根據分類層級自動創建對應的資料夾結構，每個分類都有獨立的資料夾。

輸出結果

執行完成後，您將獲得：

  1. 完整的資料夾結構：按照華藝電子書的分類體系組織
  2. .url 檔案：每個分類對應一個可直接點擊的 URL 檔案
  3. 控制台輸出：顯示已創建的檔案路徑

注意事項

  1. HTML 檔案格式：確保 all-topic.html 檔案包含完整的華藝電子書分類結構
  2. 檔案編碼：所有檔案都使用 UTF-8 編碼
  3. 權限要求：程式需要在目標目錄有寫入權限
  4. Windows 系統：.url 檔案主要在 Windows 系統上使用

技術細節

主要函數

  • sanitize_filename()：清理檔案名稱
  • create_url_file()：創建 URL 檔案
  • should_create_subcat_url()：判斷是否創建子分類 URL
  • process_category_structure()：處理完整的分類結構

依賴套件

  • BeautifulSoup4：HTML 解析
  • os：檔案系統操作
  • re：正則表達式處理

自訂化

您可以根據需要修改以下部分：

  1. URL 基礎路徑：修改 base_url 變數
  2. 檔案副檔名：修改 .url 為其他格式
  3. 資料夾結構：調整資料夾創建邏輯
  4. 過濾條件：添加更多分類過濾規則
