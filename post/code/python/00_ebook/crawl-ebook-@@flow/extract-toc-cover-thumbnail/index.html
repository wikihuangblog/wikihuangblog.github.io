電子書目錄與封面提取工具

這是一個用於從電子書網站提取目錄（TOC）、封面縮略圖和相關資訊的 Python 工具。

功能特色

  • 🔍 自動化目錄提取：從指定的電子書網站自動提取章節目錄
  • 🖼️ 封面縮略圖下載：提取並保存書籍封面圖片
  • 📁 批次處理：支援大量 .url 檔案的批次處理
  • 🧹 內容清理：自動移除不必要的圖片和樣式元素
  • 📄 HTML 輸出：將提取的內容保存為格式化的 HTML 檔案

專案結構

app/
├── app.py                          # 主要程式檔案
├── 00_temp-ready-move-area/        # 待處理的 URL 檔案存放區
│   └── urls/
│       └── 語言/                   # 按語言分類的 URL 檔案
│           ├── eng2/               # 英文書籍 URL
│           ├── 多益/               # 多益相關書籍 URL
│           └── 英文/               # 英文學習書籍 URL
├── 00_url-for-toc/                 # 用於目錄提取的 URL 檔案
│   └── urls/
├── htmlToc/                        # 輸出的 HTML 目錄檔案（執行後生成）
└── ref/                            # 參考檔案
    └── oriContent.html             # 原始網頁內容範例

安裝需求

Python 套件

pip install requests beautifulsoup4

系統需求

  • Python 3.6 或更高版本
  • Windows 作業系統（程式碼中包含 Windows 路徑格式）

使用方法

1. 準備 URL 檔案

將要處理的電子書 URL 檔案放置在 00_url-for-toc/urls/ 目錄中。URL 檔案格式如下：

[InternetShortcut]
URL=https://example.com/ebook-url

2. 執行程式

python app.py

3. 查看結果

程式執行完成後，提取的目錄和封面資訊將保存在 htmlToc/ 目錄中，每個檔案對應一個 HTML 格式的目錄頁面。

程式功能說明

主要功能

  1. URL 檔案解析：讀取 .url 檔案中的網址資訊
  2. 網頁內容下載：使用 requests 下載網頁內容
  3. HTML 解析：使用 BeautifulSoup 解析網頁結構
  4. 內容提取：
    • 書籍標題（h1.book_title）
    • 封面縮略圖（div.detail_pic_resize img）
    • 章節目錄（ul.book_chapter）
  5. 內容清理：
    • 移除 SVG 和 PNG 圖片
    • 移除 display:none 樣式
    • 清理空樣式屬性
  6. HTML 生成：將提取的內容組合成完整的 HTML 檔案

輸出格式

每個生成的 HTML 檔案包含：

  • 書籍標題
  • 封面縮略圖（如果可用）
  • 原始網址連結
  • 線上閱讀網址
  • 格式化的章節目錄

注意事項

  • 程式會自動跳過名為 repeat 的資料夾
  • 如果無法提取目錄內容，程式會顯示相應的錯誤訊息
  • 生成的 HTML 檔案使用 UTF-8 編碼
  • 程式設計為處理特定網站結構，可能需要根據目標網站調整選擇器

錯誤處理

程式包含基本的錯誤處理機制：

  • 網路連線錯誤
  • 檔案讀取錯誤
  • HTML 解析錯誤
  • 元素查找失敗

自訂設定

如需處理不同網站的電子書，可能需要修改以下部分：

  • CSS 選擇器（第 29、33、37 行）
  • 檔案路徑格式
  • 輸出 HTML 模板
