華藝電子書爬蟲專案

這是一個用於爬取華藝電子書網站書籍資訊的 Python 專案，支援自動化爬取書籍標題、URL 連結、封面圖片和目錄內容。

功能特色

  • 🚀 異步爬取：使用 Playwright 和 asyncio 實現高效異步爬取
  • 📚 多分類支援：支援按分類爬取不同類別的書籍
  • 🖼️ 圖片下載：自動下載書籍封面圖片
  • 📄 目錄爬取：可選功能，爬取書籍目錄並儲存為 HTML
  • 🔄 斷點續傳：支援記錄爬取進度，可從上次中斷處繼續
  • 📁 檔案整理：自動整理下載的檔案到對應資料夾
  • 🔍 重複檢測：自動檢測並處理重複檔案

專案結構

crawl-ebook/
├── app.py                 # 原始爬蟲腳本（舊版本）
├── main.py               # 主程式入口
├── libs/                 # 核心模組
│   ├── BookCrawler.py    # 書籍爬蟲類別
│   ├── Config.py         # 配置管理
│   └── FileManager.py    # 檔案管理
├── 華藝電子書分類/        # 分類配置目錄
│   └── 00_target/        # 目標分類 URL 檔案
├── output/               # 輸出目錄
├── 00_temp-ready-move-area/ # 臨時檔案區域
│   ├── urls/            # URL 檔案
│   └── images/          # 圖片檔案
├── toc_html/            # 目錄 HTML 檔案
└── crawler_config.json  # 爬蟲配置檔案

安裝需求

系統需求

  • Python 3.7+
  • Windows 10/11

依賴套件

pip install playwright aiohttp configparser

安裝 Playwright 瀏覽器

playwright install chromium

使用方法

1. 準備分類 URL 檔案

在 華藝電子書分類/00_target/ 目錄下放置 .url 檔案，每個檔案代表一個書籍分類。

2. 執行爬蟲

python main.py

3. 配置選項

在 libs/Config.py 中可以調整以下設定：

USE_AUTO_PAGE_CALCULATION = True  # 是否自動計算總頁數
TEST_PAGE_NUMBER = 1              # 測試模式下的固定頁數
PAGE_SIZE = 20                    # 每頁顯示的書籍數量
SORT_TYPE = 5                     # 排序方式

核心模組說明

BookCrawler

  • 負責主要的爬取邏輯
  • 支援分頁爬取
  • 可選的圖片下載功能
  • 目錄內容爬取

FileManager

  • 管理檔案儲存和整理
  • 處理檔名清理
  • 重複檔案檢測

Config

  • 管理爬蟲配置
  • 記錄爬取進度
  • 支援斷點續傳

輸出檔案說明

URL 檔案 (.url)

儲存每本書的詳細頁面連結，格式為 Windows 網際網路捷徑檔案。

圖片檔案 (.jpg)

書籍封面圖片，檔名與書籍標題對應。

目錄檔案 (.html)

書籍目錄內容，以 HTML 格式儲存，便於瀏覽。

配置檔案 (crawler_config.json)

記錄最後爬取的頁碼，用於斷點續傳功能。

注意事項

  1. 網路連線：確保網路連線穩定，爬取過程中避免中斷
  2. 目標網站：請遵守網站的使用條款，避免過於頻繁的請求
  3. 儲存空間：確保有足夠的磁碟空間儲存下載的檔案
  4. 瀏覽器模式：預設為非無頭模式，可在 BookCrawler.py 中修改為 headless=True

故障排除

常見問題

  1. Playwright 安裝失敗

    pip install --upgrade pip
pip install playwright
playwright install chromium

  2. 網路連線錯誤

    • 檢查網路連線
    • 確認目標網站可正常訪問

  3. 檔案權限錯誤

    • 確保對輸出目錄有寫入權限
    • 以管理員身份執行（如需要）

開發說明

擴展功能

  • 可在 BookCrawler.py 中添加新的爬取功能
  • 在 FileManager.py 中擴展檔案處理邏輯
  • 修改 Config.py 添加新的配置選項

程式碼結構

  • 錯誤處理機制確保程式穩定性
