📚 章節內容自動生成工具

🎯 專案概述

這是一個用於自動生成電子書章節內容的 Python 工具。該工具能夠掃描指定的目錄結構，識別章節和小節，並為每個 Markdown 檔案自動生成包含章節資訊、結構導覽和內容預覽的完整內容。

✨ 主要功能

🔍 智能章節識別

  • 自動掃描目錄結構，識別章節資料夾
  • 支援多種章節命名格式（如「第 X 章」、「ch01_UNIT-1」等）
  • 動態建立章節與小節的層級關係

📋 內容自動生成

  • 為每個 Markdown 檔案生成標準化的章節內容
  • 包含章節所屬資訊、完整章節結構導覽
  • 自動標示當前小節位置
  • 根據內容類型生成不同的說明模板

🎨 多樣化內容模板

  • 主章節：包含章節概述、學習目標
  • 習題章節：提供練習題目結構
  • Python 相關：重點概念說明
  • 一般內容：學習重點和應用範例

📁 專案結構

app/
├── app.py                 # 主程式檔案
├── 00_target/            # 目標目錄（包含原始章節結構）
├── 00_output/            # 輸出目錄（處理後的檔案）
├── 00_version/           # 版本歷史檔案
├── backuuppp/            # 備份目錄
├── ch01_UNIT-1-WHAT-IS-A-PARAGRAPH/  # 範例章節
├── python-book/          # Python 書籍章節
├── toeic/                # TOEIC 教材章節
├── goo.bat               # 批次執行腳本
├── issue.md              # 專案需求說明
└── README.md             # 專案說明文件

🚀 使用方法

基本使用

# 使用預設目標目錄 (00_target)
python app.py

# 指定自訂目標目錄
python app.py your_target_directory

批次執行

# 使用提供的批次檔
goo.bat your_target_directory

🔧 核心功能說明

1. 章節資料夾識別邏輯

程式會自動判斷哪些資料夾是章節資料夾，判斷條件：

  • 必須有兄弟資料夾（同層級有其他資料夾）
  • 必須包含多個子資料夾（至少 2 個）

2. 章節編號提取

支援多種章節編號格式：

  • 第 X 章 格式
  • 數字開頭格式
  • ch01_UNIT-1 格式

3. 內容生成模板

根據檔案類型自動選擇合適的內容模板：

主章節模板

# 章節標題

## 📚 章節所屬
**所屬章節：** 第 X 章：章節名稱

## 📋 完整章節結構
### 第 X 章：章節名稱
1. 小節一
2. 小節二 ← 當前位置
3. 小節三

## 🎯 章節概述
## 📖 學習目標

習題章節模板

## 📝 習題練習
### 練習題目
1. 基礎概念題
2. 實作練習題
3. 綜合應用題

📊 處理流程

  1. 掃描目錄結構：遞迴掃描目標目錄
  2. 識別章節：判斷哪些資料夾是章節資料夾
  3. 建立結構：建立章節與小節的層級關係
  4. 生成內容：為每個 Markdown 檔案生成內容
  5. 輸出結果：顯示處理統計和結果

🎯 使用場景

適用於：

  • 📖 電子書內容管理
  • 📚 教材章節組織
  • 🎓 課程內容結構化
  • 📝 技術文件整理

特別適合：

  • 需要為大量 Markdown 檔案添加章節資訊
  • 希望建立標準化的內容結構
  • 需要自動生成章節導覽和內容預覽

⚙️ 技術特點

  • 智能識別：自動識別章節結構，無需手動配置
  • 靈活適應：支援多種命名格式和目錄結構
  • 內容豐富：生成包含導覽、概述、目標的完整內容
  • 錯誤處理：具備完善的錯誤處理和統計功能
  • 批次處理：支援大量檔案的批次處理

📈 版本歷史

專案包含多個版本迭代：

  • v1: 基礎功能實現
  • v2: 章節判斷邏輯優化
  • v3: 兄弟資料夾檢測
  • v4: 子資料夾歸屬判斷
  • v5: 內容精簡和書籍名稱支援

🔍 範例輸出

處理完成後，每個 Markdown 檔案都會包含：

# 1.1 本書導讀

## 📚 章節所屬

**所屬章節：** 第 1 章：導讀與環境建置

**當前小節：** 1.1 本書導讀 (第 1 節)

## 📋 完整章節結構

### 第 1 章：導讀與環境建置

1. 1.1 本書導讀 ← 當前位置
2. 1.2 系統核心觀念與流程分析
3. 2.1 打造現代化開發環境
...

## 🎯 章節概述

這是第 1 章的內容。本章將介紹相關的基礎概念和實作方法。

## 📖 學習目標

- 理解本章的核心概念
- 掌握相關的實作技巧
- 能夠應用所學知識解決問題

🛠️ 系統需求

  • Python 3.6+
  • 無需額外依賴套件（僅使用標準函式庫）

📝 注意事項

  1. 確保目標目錄存在且具有讀寫權限
  2. 程式會覆蓋現有的 Markdown 檔案內容
  3. 建議在執行前備份重要檔案
  4. 支援 UTF-8 編碼的中文內容
