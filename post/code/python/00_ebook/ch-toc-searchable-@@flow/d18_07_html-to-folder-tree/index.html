HTML 目錄轉資料夾結構工具

這是一個 Python 應用程式，用於將 HTML 格式的目錄結構轉換為實際的資料夾層次結構，並為每個章節創建 PDF 頁面捷徑。

功能特色

  • 📁 解析 HTML 目錄結構（<ul> 和 <li> 標籤）
  • 🗂️ 自動創建對應的資料夾層次結構
  • 🔢 為每個資料夾添加層次編號前綴（如 01.00.00_）
  • 🔗 為每個章節創建 PDF 頁面捷徑（.lnk 檔案）
  • 🧹 自動清理資料夾名稱，移除 Windows 不允許的字符
  • 📖 支援多層巢狀目錄結構

專案結構

app/
├── app.py                    # 主要應用程式檔案
├── Coding-Interviews.htm     # 輸入的 HTML 目錄檔案
├── folder/                   # 生成的資料夾結構
│   ├── 00_Cover/
│   ├── 01_Chapter-1_-Interview-Process/
│   ├── 02_Chapter-2_-Programming-Languages/
│   └── ...
├── 00_issue/                 # 問題描述
├── 00_ref/                   # 參考檔案
└── README.md                 # 本說明檔案

系統需求

  • Python 3.6+
  • Windows 作業系統（用於創建 .lnk 捷徑）
  • 以下 Python 套件：
    • beautifulsoup4 - HTML 解析
    • pywin32 - Windows COM 介面

安裝步驟

  1. 確保已安裝 Python 3.6 或更高版本
  2. 安裝必要的套件：
pip install beautifulsoup4 pywin32

使用方法

  1. 將您的 HTML 目錄檔案命名為 Coding-Interviews.htm 並放在專案根目錄
  2. 在 app.py 中修改 PDF 檔案路徑（第 115 行）： 
    pdf_path = r"您的PDF檔案路徑"
  3. 執行程式： 
    python app.py

輸出結果

程式會：

  1. 在 folder/ 目錄下創建對應的資料夾結構
  2. 每個資料夾名稱格式為：層次編號_清理後的章節名稱
  3. 在每個資料夾中創建對應 PDF 頁面的捷徑檔案（.lnk）

範例輸出結構

folder/
├── 00.00_Contents-at-a-Glance/
│   └── 3.lnk
├── 00.01_Table-of-Contents/
│   └── 287.lnk
├── 01.00.00_Phone-Interviews/
│   └── 6.lnk
└── 01.00.01_On-Site-Interviews/
    └── 8.lnk

程式碼說明

主要功能

  • clean_folder_name(): 清理資料夾名稱，移除非法字符
  • create_pdf_shortcut(): 創建 PDF 頁面捷徑
  • Node 類別: 表示目錄樹節點
  • build_tree(): 建構目錄樹結構
  • create_folder_structure(): 創建實際的資料夾結構

HTML 格式要求

輸入的 HTML 檔案應包含巢狀的 <ul> 和 <li> 結構，每個 <li> 包含一個 <a> 標籤，其中：

  • href 屬性包含 PDF 頁面資訊（如 #page=123）
  • 文字內容為章節標題

注意事項

  • 此工具專為 Windows 系統設計，使用 Windows COM 介面創建捷徑
  • 需要安裝 Adobe Acrobat DC 或修改捷徑目標路徑
  • 資料夾名稱會自動清理，移除 Windows 不允許的字符
  • 程式會自動跳過空名稱或無效的目錄項目

自訂設定

您可以修改以下設定：

  1. PDF 路徑（第 115 行）：

    pdf_path = r"您的PDF檔案完整路徑"

  2. 輸出目錄（第 122 行）：

    output_dir = "您的輸出目錄名稱"

  3. Adobe Acrobat 路徑（第 43 行）：

    shortcut.Targetpath = r"您的Acrobat執行檔路徑"

故障排除

  • ImportError: 確保已安裝所有必要的套件
  • COM 錯誤: 確保在 Windows 系統上執行
  • PDF 捷徑無法開啟: 檢查 Adobe Acrobat 安裝路徑和 PDF 檔案路徑
  • 資料夾創建失敗: 檢查檔案權限和磁碟空間
