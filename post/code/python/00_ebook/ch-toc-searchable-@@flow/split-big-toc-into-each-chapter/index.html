電子書目錄分割工具

這是一個用於處理電子書目錄的 Python 工具，可以將大型的目錄文件自動分割成各個章節的獨立 Markdown 文件，並進行格式化處理。

功能特色

  • 📚 自動章節分割：將大型目錄文件按章節自動分割
  • 🔧 智能格式化：自動處理標題格式、編號和空格
  • 📁 結構化組織：為每個章節創建獨立的資料夾和文件
  • 🏷️ 標準化命名：使用統一的章節編號格式（ch01, ch02…）
  • 🔄 批量處理：一次性處理整個目錄結構

項目結構

app/
├── app.py                 # 主程序文件
├── toc.md                # 原始目錄文件
├── toc_modified.md       # 格式化後的目錄文件
├── chapter/              # 章節文件輸出目錄
├── 00_issue/            # 問題記錄目錄
│   └── issue.md
├── ref/                 # 參考文件目錄
│   └── format-toc.py
└── run.bat              # Windows 批次執行文件

安裝與使用

系統需求

  • Python 3.6 或更高版本
  • Windows 系統（支援批次文件執行）

使用方法

方法一：使用批次文件（推薦）

# 在 Windows 命令提示字元中執行
run.bat toc.md

方法二：直接執行 Python 腳本

# 在專案根目錄執行
python app.py toc.md

參數說明

  • toc.md：輸入的目錄文件路徑（必需參數）

處理流程

  1. 讀取輸入文件：載入指定的目錄文件
  2. 格式化處理：
    • 將中文括號 （） 轉換為英文括號 ()
    • 處理標題層級和編號格式
    • 統一章節編號格式為 ch01_, ch02_ 等
  3. 生成格式化文件：保存為 toc_modified.md
  4. 分割章節：
    • 按 ### 標記分割章節
    • 為每個章節創建獨立資料夾
    • 生成對應的 Markdown 文件

輸出結果

執行完成後，會在 chapter/ 目錄下生成以下結構：

chapter/
├── ch01_Interview Process/
│   └── ch01_Interview Process.md
├── ch02_Programming Languages/
│   └── ch02_Programming Languages.md
├── ch03_Data Structures/
│   └── ch03_Data Structures.md
└── ...

範例

輸入文件格式（toc.md）

### Contents at a Glance
### About the Author
### Introduction
- **Chapter 1: Interview Process**
    - Types of Interviews
    - Phases of Interviews
- **Chapter 2: Programming Languages**
    - C++
    - Java

輸出結果

  • 格式化文件：toc_modified.md
  • 章節文件：chapter/ch01_Interview Process/ch01_Interview Process.md

錯誤處理

程序包含完整的錯誤處理機制：

  • 檢查輸入文件是否存在
  • 驗證文件編碼格式
  • 處理文件讀寫權限問題
  • 提供詳細的錯誤訊息

技術細節

主要功能模組

  1. modify_text()：文本格式化處理
  2. create_chapter_files()：章節文件創建
  3. main()：主程序流程控制

支援的格式

  • Markdown 標題格式（#、##、### 等）
  • 多層級列表結構
  • 中文和英文混合內容
  • 各種編號格式

注意事項

  • 確保輸入文件使用 UTF-8 編碼
  • 章節標題需要以 ### 開頭才能正確分割
  • 程序會自動創建必要的目錄結構
  • 重複執行會覆蓋已存在的文件
