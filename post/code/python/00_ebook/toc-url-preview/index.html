TOC URL Preview 工具

這是一個用於將目錄（Table of Contents）轉換為 Google 搜尋 URL 預覽文件的工具。當您有一個完整的章節目錄時，這個工具可以幫助您快速生成所有章節的 Google 搜尋結果預覽，讓您能夠快速瀏覽相關資訊。

功能特色

  • 📚 自動解析目錄文件（toc.txt）
  • 🔍 為每個章節生成 Google 搜尋 URL
  • 📁 按章節自動分類到對應資料夾
  • 🖥️ 支援 Windows .url 文件格式
  • 🔗 支援 Windows .lnk 捷徑文件格式
  • 🐍 提供 Python 腳本自動化處理
  • 💻 提供 PowerShell 腳本手動創建

專案結構

app/
├── ch01/                    # 章節資料夾（自動生成）
│   └── 01_Flutter_簡介.url
├── ok/                      # 範例輸出資料夾
│   ├── GoogleNumpy.lnk
│   └── GoogleNumpy.url
├── toc.txt                  # 目錄文件
├── url-create.py           # Python 自動化腳本
├── url-create.ps1          # PowerShell URL 創建腳本
├── lnk-create.ps1          # PowerShell 捷徑創建腳本
└── memo.markdown           # 專案說明文件

使用方法

方法一：使用 Python 腳本（推薦）

  1. 準備您的目錄文件 toc.txt，格式如下：

    01_Flutter_簡介
1.1_Flutter_的優勢
1.2_比較其他技術
02_Dart_語言入門
2.1_應用場景
...

  2. 執行 Python 腳本：

    python url-create.py

  3. 腳本會自動：

    • 讀取 toc.txt 文件
    • 為每個章節創建對應的資料夾（ch01, ch02, …）
    • 生成 Google 搜尋 URL 文件（.url 格式）

方法二：使用 PowerShell 腳本

創建 URL 文件

.\url-create.ps1

創建捷徑文件

.\lnk-create.ps1

文件格式說明

.url 文件格式

[InternetShortcut]
URL=https://www.google.com/search?q=搜尋關鍵字

.lnk 文件格式

Windows 捷徑文件，可以直接雙擊開啟瀏覽器並執行搜尋。

使用場景

這個工具特別適合以下情況：

  • 📖 學習新技術時，需要快速了解各個章節的相關資訊
  • 🔍 研究主題時，想要一次性獲得所有相關章節的搜尋結果
  • 📚 準備教學材料時，需要收集各章節的參考資料
  • 🎯 快速預覽大量相關資訊，然後選擇性地深入學習

技術特點

  • 自動化處理：一鍵將整個目錄轉換為搜尋預覽
  • 智能分類：根據章節編號自動創建對應資料夾
  • 編碼兼容：支援 UTF-8 和 Latin-1 編碼
  • 跨平台：Python 腳本可在多個平台運行
  • Windows 優化：PowerShell 腳本專為 Windows 環境優化

注意事項

  • 確保 toc.txt 文件存在於腳本同目錄下
  • 目錄格式應遵循：章節編號_章節名稱
  • 生成的 URL 文件會自動編碼中文字符
  • 建議在執行前備份重要文件

系統需求

  • Python 3.x（用於 Python 腳本）
  • Windows PowerShell（用於 PowerShell 腳本）
  • 網路連接（用於 Google 搜尋）

授權

此專案僅供學習和研究使用。請遵守 Google 搜尋服務的使用條款。
