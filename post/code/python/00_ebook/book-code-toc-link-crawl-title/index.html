書籍 URL 爬蟲與快捷方式生成器

這是一個用於爬取旗標出版社書籍資訊並生成 URL 快捷方式的 Python 專案。

專案概述

本專案包含三個主要腳本，用於處理旗標出版社的書籍編號，並生成對應的 URL 快捷方式文件：

  1. app.py - 基本版本，直接生成 URL 快捷方式
  2. v5_title-not-get-content-instead.py - 進階版本，爬取網頁標題並生成帶標題的快捷方式
  3. v6_zip-url.py - 專門用於生成 ZIP 下載連結的快捷方式

檔案結構

app/
├── app.py                              # 基本 URL 生成器
├── v5_title-not-get-content-instead.py # 帶標題爬蟲版本
├── v6_zip-url.py                      # ZIP 下載連結生成器
├── number.txt                         # 書籍編號列表
├── urls/                              # 生成的 URL 快捷方式目錄
└── README.md                          # 專案說明文件

功能特色

app.py

  • 讀取 number.txt 中的書籍編號
  • 生成旗標出版社產品頁面的 URL 快捷方式
  • 將所有快捷方式保存到 urls/ 目錄

v5_title-not-get-content-instead.py

  • 爬取每個書籍頁面的標題
  • 為每個書籍編號創建專屬資料夾
  • 生成帶有書籍標題的 URL 快捷方式
  • 包含錯誤處理和日誌記錄功能

v6_zip-url.py

  • 生成 ZIP 檔案下載連結的快捷方式
  • 使用固定的 “zipRef” 標題
  • 適用於直接下載書籍相關檔案

安裝需求

pip install requests beautifulsoup4

使用方法

1. 準備書籍編號

在 number.txt 文件中輸入書籍編號，每行一個：

F1005
F2035
F2005
...

2. 執行腳本

基本版本（app.py）

python app.py

進階版本（v5_title-not-get-content-instead.py）

python v5_title-not-get-content-instead.py

ZIP 連結版本（v6_zip-url.py）

python v6_zip-url.py

輸出結果

app.py 輸出

  • 在 urls/ 目錄下生成 {編號}.url 檔案
  • 每個檔案包含對應的產品頁面連結

v5_title-not-get-content-instead.py 輸出

  • 在 urls/{編號}/ 目錄下生成 {編號}_{標題}.url 檔案
  • 包含完整的書籍標題資訊

v6_zip-url.py 輸出

  • 在 urls/{編號}/ 目錄下生成 {編號}_zipRef.url 檔案
  • 直接連結到 ZIP 下載頁面

URL 格式

  • 產品頁面: https://www.flag.com.tw/books/product/{編號}
  • ZIP 下載: https://dl.flag.com.tw/download/{編號}.zip

錯誤處理

  • v5 和 v6 版本包含完整的錯誤處理機制
  • 錯誤訊息會記錄到 error.log 檔案
  • 網路請求失敗時會顯示錯誤訊息並繼續處理其他編號

注意事項

  1. 確保網路連線正常，以便爬取網頁標題
  2. 大量請求時請注意網站的使用條款
  3. 生成的 .url 檔案可在 Windows 系統中直接使用
  4. 建議在執行前備份重要資料
