TOKO OCR 圖片文字識別工具

這是一個基於 Python 和 Tesseract OCR 的圖片文字識別工具，可以將圖片中的文字提取出來並轉換為 Markdown 格式。

功能特色

  • 支援多種圖片格式（JPG、PNG 等）
  • 使用 Tesseract OCR 引擎進行文字識別
  • 支援繁體中文識別
  • 自動將識別結果輸出為 Markdown 格式
  • 批次處理資料夾中的所有圖片

系統需求

  • Python 3.6+
  • Tesseract OCR
  • 相關 Python 套件

安裝步驟

1. 安裝 Tesseract OCR

Windows

  1. 下載並安裝 Tesseract OCR
  2. 預設安裝路徑：C:\Program Files\Tesseract-OCR\tesseract.exe

macOS

brew install tesseract

Ubuntu/Debian

sudo apt-get install tesseract-ocr

2. 安裝 Python 套件

pip install pytesseract pillow

使用方法

基本用法

python app.py <輸入資料夾路徑>

範例

# 處理 input 資料夾中的圖片
python app.py input

# 處理 v1 資料夾中的圖片
python app.py input/v1

專案結構

toko-ocr/
├── app.py                 # 主程式
├── input/                 # 輸入資料夾
│   ├── input.md          # 輸出結果
│   ├── *.jpg             # 待處理的圖片
│   └── v1/               # 子資料夾
│       ├── input.md      # 輸出結果
│       └── *.jpg         # 待處理的圖片
└── README.md             # 說明文件

程式碼說明

主要類別：ImageOCRProcessor

  • __init__(input_folder, languages, tesseract_cmd): 初始化 OCR 處理器
  • ocr_image(image_path, language): 對單一圖片進行 OCR 識別
  • process_images(): 批次處理資料夾中的所有圖片

設定參數

  • languages: 識別語言，預設為 ['chi_tra']（繁體中文）
  • tesseract_cmd: Tesseract 執行檔路徑

輸出格式

程式會自動在輸入資料夾中生成一個與資料夾同名的 .md 檔案，包含所有圖片的 OCR 識別結果。

注意事項

  1. 確保 Tesseract OCR 已正確安裝並設定路徑
  2. 圖片品質會影響 OCR 識別準確度
  3. 目前主要針對繁體中文進行優化
  4. 支援的圖片格式：JPG、PNG 等常見格式

故障排除

常見問題

  1. 找不到 tesseract 執行檔

    • 確認 Tesseract 已正確安裝
    • 檢查 tesseract_cmd 路徑是否正確

  2. OCR 識別準確度低

    • 確保圖片清晰度足夠
    • 嘗試調整圖片對比度和亮度
    • 考慮使用更高解析度的圖片

  3. 中文識別問題

    • 確認已安裝中文語言包
    • 檢查 languages 參數設定
