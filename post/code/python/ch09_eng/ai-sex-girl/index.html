AI 角色聊天工具

這是一個用於處理 Yodayo.com 網站上 AI 角色聊天連結的 Python 工具。該工具可以從 HTML 文件中提取角色資訊，建立資料夾結構，並下載相關圖片。

功能特色

  • 🔗 自動提取連結：從 HTML 文件中提取 /tavern/chat/ 開頭的連結
  • 📁 智能資料夾管理：根據角色名稱自動建立資料夾，避免重複
  • 🖼️ 圖片下載：自動下載角色的封面圖片
  • 🔗 快捷方式建立：為每個角色建立 Opera 瀏覽器快捷方式
  • 🧹 重複內容處理：合併多個 txt 文件並移除重複的連結
  • 🛡️ Windows 路徑安全：自動處理 Windows 不支援的字元

專案結構

app/
├── app.py                 # 主程式
├── simple_test.py         # 測試檔案
├── ori.html              # 原始 HTML 檔案
├── issue.txt             # 問題記錄
├── src/                  # 來源 txt 檔案資料夾
│   ├── 1.txt
│   ├── 2.txt
│   └── ...
├── 00_output/            # 輸出資料夾（自動建立）
├── 00_version/           # 版本歷史
│   ├── v1_url-ok;wait-open-by-opera.py
│   ├── v2_opera-lnk-ok;wait-webp.py
│   ├── v3_webp-ok;wait-all-txt-combine.py
│   ├── v4_txt-combine-ok;wait-log-fail;wait-remove-repeat.py
│   └── v5_no-repeat-ok;trim-wrong-name.py
└── backuupp/             # 備份資料夾

安裝需求

  • Python 3.6+
  • Windows 作業系統（用於建立 .lnk 快捷方式）
  • Opera 瀏覽器（可選，用於快捷方式）

使用方法

1. 基本使用

python app.py

2. 配置設定

在 app.py 中修改以下設定：

DOMAIN = "https://yodayo.com/"                    # 目標網站
HTML_FILE = "ori.html"                           # HTML 檔案名稱
OUTPUT_ROOT = "00_output"                        # 輸出根目錄
SRC_FOLDER = "src"                               # 來源資料夾
OPERA_PATH = r"C:\Users\zero\AppData\Local\Programs\Opera\opera.exe"  # Opera 路徑

3. 資料來源

程式會按以下優先順序尋找資料：

  1. src 資料夾：合併所有 .txt 檔案中的內容
  2. ori.html：如果 src 資料夾不存在或為空

主要功能說明

1. 連結提取 (extract_entries)

從 HTML 內容中提取包含以下資訊的條目：

  • 角色名稱（從 <span> 標籤）
  • 聊天連結（href="/tavern/chat/..."）
  • 封面圖片 URL

2. 檔案名稱清理 (sanitize_for_windows_path)

自動處理 Windows 不支援的字元：

  • 移除或替換特殊字元：<>:"/\|?*
  • 處理 PowerShell 問題字元：'$(){}[];|&<>#@!%^+=~`
  • 限制檔案名稱長度
  • 避免空名稱

3. 重複內容處理 (combine_txt_files_to_html)

  • 合併 src 資料夾中的所有 .txt 檔案
  • 移除重複的 <a> 標籤
  • 轉換為標準 HTML 格式

4. 快捷方式建立 (create_opera_shortcut)

使用 PowerShell 建立 .lnk 快捷方式：

  • 目標：Opera 瀏覽器
  • 參數：完整的聊天 URL
  • 自動處理路徑中的特殊字元

版本歷史

  • v1: 基本 URL 提取功能
  • v2: 加入 Opera 快捷方式支援
  • v3: 支援 WebP 圖片格式
  • v4: 加入 txt 檔案合併和重複移除
  • v5: 改善檔案名稱處理，解決特殊字元問題

常見問題

1. 快捷方式建立失敗

問題：PowerShell 錯誤，通常是因為檔案名稱包含特殊字元

解決方案：

  • 程式已內建 sanitize_for_windows_path 函數自動處理
  • 如果仍有問題，檢查 Opera 路徑是否正確

2. 圖片下載失敗

可能原因：

  • 網路連線問題
  • 圖片 URL 無效
  • 權限問題

3. 資料夾名稱問題

程式會自動處理以下情況：

  • 重複的資料夾名稱（自動加上編號）
  • 特殊字元替換
  • 過長的名稱截斷

輸出結果

執行成功後，會在 00_output 資料夾中建立：

00_output/
├── 角色名稱1/
│   ├── yodayo.lnk        # Opera 快捷方式
│   └── cover.jpg         # 封面圖片
├── 角色名稱2/
│   ├── yodayo.lnk
│   └── cover.webp
└── ...

技術細節

  • 檔案處理：支援 UTF-8 編碼
  • 錯誤處理：每個步驟都有異常處理機制
  • 跨平台考量：主要針對 Windows 系統優化
