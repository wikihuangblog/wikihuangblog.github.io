Markdown 翻譯工具

這是一個用於將英文 Markdown 文件翻譯成繁體中文並生成 HTML 格式輸出的 Python 工具。

功能特色

  • 🔄 自動翻譯：使用 Google Translate API 將英文內容翻譯成繁體中文
  • 📝 Markdown 支援：支援 Markdown 格式的文本處理
  • 🎨 HTML 輸出：生成美觀的 HTML 文件，包含原文和譯文對照
  • ⚡ 智能分塊：自動將長文本分割成適當大小的塊進行翻譯
  • 🎯 特殊處理：保留特殊詞彙（如人名）不被翻譯
  • 📊 API 統計：顯示翻譯過程中的 API 調用次數

系統需求

  • Python 3.6+
  • 網路連線（用於 Google Translate API）

安裝依賴

pip install googletrans==4.0.0rc1

使用方法

方法一：使用批次檔（推薦）

# Windows
run.bat "輸入文件路徑"

# 範例
run.bat "memo.md"

方法二：直接執行 Python 腳本

python app.py "輸入文件路徑"

# 範例
python app.py "memo.md"

輸出格式

程式會生成與輸入文件同名的 HTML 文件，包含：

  • 翻譯後的中文內容
  • 原始英文內容
  • 美觀的 HTML 格式排版
  • 特殊詞彙的保留處理

版本演進

v1 - 基本翻譯功能

  • 實現基本的英文到繁體中文翻譯
  • 簡單的文本分割和處理

v2 - 合併調用優化

  • 優化 API 調用次數
  • 改進文本分割邏輯

v3 - 表格處理

  • 移除表格行分割問題
  • 改善格式化處理

v4 - 大文件支援

  • 支援大文件翻譯
  • 改進記憶體使用效率

v5 - 特殊名稱處理

  • 排除特殊名稱（如人名）不被翻譯
  • 添加 TTS 相關功能

  • 支援命令行參數輸入
  • 改進錯誤處理

技術特點

翻譯類別 (Translator)

  • split_text(): 智能文本分塊，避免超過 API 限制
  • translate(): 執行翻譯並統計 API 調用次數

格式化功能

  • format_bold(): 處理 Markdown 粗體格式
  • translate_and_format_html(): 生成 HTML 輸出

特殊處理

  • 保留特殊詞彙（如 “Iris”, “wei”）不被翻譯
  • 使用 aria-hidden 屬性標記特殊內容
  • 支援 HTML 轉義處理

使用範例

輸入文件 (memo.md)

*You're relaxing at home, reminiscing about your school day...*

"Hey, are you by chance wei?" *She says with a **stern** tone*

輸出文件 (memo.html)

<html><head><meta charset='utf-8'></head><body>
<p aria-hidden='true'>*你在家裡放鬆，回憶著你的學校生活...*</p>
<p>*You're relaxing at home, reminiscing about your school day...*</p>
<p aria-hidden='true'>「嘿，你碰巧是wei嗎？」*她用**嚴厲**的語調說道*</p>
<p>"Hey, are you by chance wei?" *She says with a **stern** tone*</p>
</body></html>

注意事項

  1. 網路連線：需要穩定的網路連線以使用 Google Translate API
  2. API 限制：Google Translate 有使用頻率限制，大量翻譯時請注意
  3. 文件編碼：輸入文件必須使用 UTF-8 編碼
  4. 特殊詞彙：可在程式碼中修改 special_words 列表來添加需要保留的詞彙

錯誤處理

程式包含完整的錯誤處理機制：

  • 文件不存在錯誤
  • 編碼錯誤
  • 網路連線錯誤
  • API 調用錯誤
