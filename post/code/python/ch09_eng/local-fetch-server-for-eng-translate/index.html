本地翻譯伺服器 (Local Translation Server)

這是一個基於 Flask 的本地翻譯伺服器，專門為 Tampermonkey 腳本提供英文到繁體中文的翻譯服務。主要用於 JanitorAI 聊天頁面的即時翻譯功能。

功能特色

  • 🌐 本地翻譯服務：使用 Google Translate API 提供翻譯服務
  • 🔄 批量句子翻譯：支援多句子同時翻譯，使用 ||| 分隔符
  • 🎯 中英對照顯示：翻譯結果以中英對照格式呈現
  • ⌨️ 快捷鍵操作：支援 F8 快捷鍵觸發翻譯
  • 🚀 CORS 支援：允許跨域請求，配合 Tampermonkey 腳本使用

專案結構

app/
├── app.py                                    # 主要翻譯伺服器 (支援句子分割)
├── Virtuoso Final List Item Translate...js  # Tampermonkey 腳本
├── 00_version---py/                          # Python 版本歷史
│   ├── v1_only-single-string;wait-split-multi/
│   └── v2_long-string-to-split.py
├── 00_version--js/                           # JavaScript 版本歷史
│   ├── translate-final-section/
│   ├── translate-v2/
│   └── translate-v3/
└── 00_readme/                               # 說明文件

安裝與設定

1. 安裝 Python 依賴

pip install flask googletrans==4.0.0-rc1 flask-cors

2. 啟動翻譯伺服器

python app.py

伺服器將在 http://localhost:5008 啟動

3. 安裝 Tampermonkey 腳本

  1. 安裝 Tampermonkey 瀏覽器擴展
  2. 將 Virtuoso Final List Item Translate (Google API, 中英對照)-1.2 .user.js 腳本添加到 Tampermonkey
  3. 確保腳本在 JanitorAI 聊天頁面 (https://janitorai.com/chats/1059445479) 上啟用

使用方法

翻譯伺服器 API

端點: POST /translate

請求格式:

{
    "text": "Hello world. How are you?|||I am fine.",
    "dest": "zh-tw"
}

回應格式:

{
    "translatedText": "你好世界。你好嗎？|||我很好。"
}

Tampermonkey 腳本使用

  1. 在 JanitorAI 聊天頁面中，找到最後一個 list item
  2. 按下 F8 快捷鍵觸發翻譯
  3. 腳本會自動將英文內容翻譯成繁體中文，並以中英對照格式顯示

版本歷史

Python 版本

  • v1: 單一字符串翻譯，等待分割多句子功能
  • v2: 支援長字符串分割翻譯
  • 當前版本: 完整支援句子分割和批量翻譯

JavaScript 版本

  • v1: 線上 JS API (需要付費)
  • v2: 使用 Google API 的中英對照翻譯
  • v3: 簡化版翻譯腳本
  • v4: 整合 Python 後端的完整解決方案

技術細節

翻譯流程

  1. Tampermonkey 腳本偵測頁面中的文本內容
  2. 將文本按句子分割，使用 ||| 連接
  3. 發送 POST 請求到本地翻譯伺服器
  4. 伺服器使用 Google Translate API 進行翻譯
  5. 返回翻譯結果，腳本更新頁面顯示

支援的語言

  • 預設目標語言：繁體中文 (zh-tw)
  • 可透過 API 參數調整目標語言

故障排除

常見問題

  1. 翻譯伺服器無法啟動

    • 檢查 Python 依賴是否正確安裝
    • 確認端口 5008 未被其他程序佔用

  2. Tampermonkey 腳本無法翻譯

    • 確認翻譯伺服器正在運行
    • 檢查瀏覽器控制台是否有錯誤訊息
    • 確認腳本在正確的頁面上啟用

  3. 翻譯結果不正確

    • 檢查 Google Translate API 的網路連接
    • 確認文本格式正確（使用 ||| 分隔句子）

開發說明

本地開發

# 啟動開發伺服器
python app.py

# 伺服器將在 debug 模式下運行
# 日誌會顯示詳細的翻譯請求資訊

自定義設定

  • 修改 app.py 中的端口號（預設：5008）
  • 調整目標語言（預設：zh-tw）
  • 自定義句子分隔符（預設：|||）
