HTML 英文翻譯工具

這是一個用於將 HTML 文件中的英文內容翻譯為繁體中文的 Python 應用程式。主要用於處理課程轉錄文件，將英文內容翻譯並生成雙語對照的 HTML 文件。

功能特色

  • 🔄 自動翻譯 HTML 文件中的英文內容為繁體中文
  • 📝 支援課程轉錄文件的特殊格式處理
  • 🎨 生成雙語對照的 HTML 輸出
  • 📦 批次處理多個 HTML 文件
  • ⚡ 智能文本分塊，避免翻譯長度限制
  • 🎯 使用 Google Translate API 進行高品質翻譯

系統需求

  • Python 3.6 或更高版本
  • 網路連線（用於 Google Translate API）

安裝依賴

pip install beautifulsoup4 googletrans==4.0.0rc1 htmlmin

使用方法

方法一：使用批次檔（推薦）

  1. 將 go.bat 文件複製到包含 HTML 文件的目標資料夾
  2. 雙擊執行 go.bat
  3. 程式會自動處理該資料夾中的所有 .html 文件

方法二：使用命令列

python app.py <目標資料夾路徑>

例如：

python app.py ./target

文件結構

app/
├── app.py              # 主要程式檔案
├── go.bat              # Windows 批次執行檔
├── target/             # 目標 HTML 文件資料夾
│   └── *.html         # 待翻譯的 HTML 文件
└── README.md          # 說明文件

程式功能說明

HTMLTranslator 類別

  • __init__(target_folder): 初始化翻譯器，設定目標資料夾
  • split_text(text, max_length=4500): 將長文本分割成適合翻譯的小塊
  • translate_text(text): 使用 Google Translate 翻譯文本
  • translate_html_file(file_path): 翻譯單個 HTML 文件
  • process_files(): 批次處理資料夾中的所有 HTML 文件

支援的 HTML 結構

程式會自動識別以下 HTML 結構：

  • h3.classroom-transcript__title - 課程標題
  • div.classroom-transcript__lines > p - 主要段落
  • div.classroom-transcript__lines > span - 備用段落格式
  • div.transcripts-container > p > span - 另一種段落格式

輸出格式

翻譯後的 HTML 文件會包含：

  • 原始課程標題
  • 雙語對照內容（英文原文 + 繁體中文翻譯）
  • 引用外部 CSS 樣式表進行美化

注意事項

  1. 網路連線：需要穩定的網路連線以使用 Google Translate API
  2. 文件備份：程式會直接覆蓋原始 HTML 文件，建議先備份
  3. 翻譯品質：翻譯品質依賴 Google Translate，可能會有誤差
  4. CSS 樣式：輸出文件會引用外部 CSS 文件，請確保路徑正確

錯誤處理

  • 如果目標資料夾不存在，程式會顯示錯誤訊息
  • 如果 HTML 文件格式不符合預期，程式會跳過該文件
  • 網路連線問題會導致翻譯失敗

自訂設定

您可以修改以下參數：

  • max_length：文本分塊的最大長度（預設 4500 字元）
  • CSS 文件路徑：在 app.py 第 67 行修改
  • 翻譯語言：在 app.py 第 32 行修改目標語言
