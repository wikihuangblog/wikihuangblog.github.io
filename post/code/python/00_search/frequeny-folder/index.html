資料夾頻率分析工具 (Frequency Folder Analyzer)

專案簡介

這是一個用於分析資料夾使用頻率的 Python 工具，主要用於統計和分析不同類型資料夾的出現頻率，並生成具有可點擊連結的 Markdown 報告。

功能特色

  • 📊 智能路徑分析：自動識別和分類不同類型的資料夾路徑
  • 🎯 分類統計：分別統計 CodeALL、ProjProj、Sidebar 三種類型的資料夾
  • 📝 Markdown 輸出：生成具有可點擊連結的 Markdown 格式報告
  • 🔍 路徑過濾：智能過濾無意義的資料夾名稱（如 output、projproj、dist 等）
  • 📏 路徑長度限制：根據不同類型設定最小路徑長度要求

使用方法

基本使用

  1. 將要分析的資料檔案放在 src/ 資料夾中
  2. 修改 app.py 中的檔案路徑： 
    file_path = "src/0224.txt"  # 更改為您的檔案名稱
  3. 執行程式： 
    python app.py
  4. 查看生成的 results.md 檔案

分析規則

程式會根據以下規則進行分析：

資料夾類型分類

  • CodeALL 資料夾：包含 “codeAll” 的路徑
  • ProjProj 資料夾：包含 “projproj” 的路徑
  • Sidebar 資料夾：包含 “sidebar” 的路徑

路徑長度要求

  • CodeALL：至少 5 層路徑
  • ProjProj：至少 7 層路徑
  • Sidebar：至少 7 層路徑

無意義資料夾過濾

自動過濾以下無意義的資料夾名稱，並向上尋找有意義的父資料夾：

  • output
  • projproj
  • dist
  • build
  • temp
  • tmp
  • app

技術細節

正則表達式

folder_pattern = re.compile(r"[A-Z]:\\(?:[^\\\n]+\\)+")

用於匹配 Windows 格式的資料夾路徑。

核心演算法

  1. 使用正則表達式提取資料夾路徑
  2. 智能過濾無意義的資料夾名稱
  3. 根據路徑內容進行分類
  4. 使用 Counter 進行頻率統計
  5. 生成 Markdown 格式的報告

需求說明

根據 issue.md 的需求，此工具主要解決以下問題：

  1. 分析資料夾使用頻率：了解哪些資料夾最常被使用
  2. 分類統計：分別統計不同類型資料夾的使用情況
  3. 生成可點擊報告：輸出具有連結功能的 Markdown 報告
  4. 智能路徑處理：處理 .url 檔案並取得父資料夾路徑

注意事項

  • 確保輸入檔案使用 UTF-8 編碼
  • 路徑格式必須為 Windows 格式（如：E:\path\to\folder）
  • 程式會自動過濾路徑長度不足的資料夾
  • 生成的報告會覆蓋現有的 results.md 檔案
