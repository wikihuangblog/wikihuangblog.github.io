Everything 按存取日期排序搜尋工具

這是一個使用 Everything 搜尋工具來按檔案存取日期排序搜尋特定檔案的專案。

專案概述

本專案提供了一個批次檔案腳本，使用 Everything 搜尋工具來搜尋指定路徑下的檔案，並按照存取日期進行排序。主要用於快速找到最近存取的應用程式檔案。

功能特色

  • 🔍 使用 Everything 搜尋工具進行高效檔案搜尋
  • 📅 按存取日期排序（最新的在前）
  • 🎯 支援正規表達式檔案名稱過濾
  • 📁 可排除特定目錄
  • ⏰ 支援日期範圍過濾
  • 🔧 可配置的搜尋參數

檔案結構

app/
├── 00_version/                    # 版本歷史檔案
│   ├── app___original___.lnk      # 原始應用程式連結
│   ├── v1_filter-ok;wait-sort-by-data-access.bat
│   ├── v2_sory-by-da-ok;wait-para-readable.bat
│   ├── v3-fail.bat
│   ├── v4_para.bat
│   ├── v5_py-ps1.bat
│   └── v6_date-format-from-year.bat
├── pic/                          # 圖片資源
│   └── image-20250302121402797.png
├── ref/                          # 參考檔案
│   └── statistic.bat
├── issue.markdown                # 問題記錄
└── recent-app.bat               # 主要執行檔案

使用方法

基本使用

  1. 確保已安裝 Everything 搜尋工具
  2. 執行 recent-app.bat 檔案
  3. 系統會自動開啟 Everything 並顯示搜尋結果

配置參數

在 recent-app.bat 中可以修改以下參數：

set EVERYTHING_PATH="C:\Program Files\Everything\Everything.exe"  # Everything 安裝路徑
set CONFIG_filter=-filter "everything"                           # 使用 everything 過濾器
set SEARCH_PATH=E:\codeAll\                                      # 搜尋路徑
set SEARCH_exclude="!\00_version\"                               # 排除的目錄
set SEARCH_PATTERN=regex:""""app\.(py^|ps1^|jpeg)$""""          # 檔案名稱模式
set DATE_RANGE=dm:2025/04/01..2025/06/30                        # 日期範圍

搜尋模式說明

  • 檔案類型過濾：regex:""""app\.(py^|ps1^|jpeg)$""""
    • 搜尋以 “app.” 開頭，副檔名為 py、ps1 或 jpeg 的檔案
  • 日期範圍：dm:2025/04/01..2025/06/30
    • 搜尋 2025年4月1日 到 2025年6月30日 之間存取的檔案
  • 排序方式：-sort "Date Accessed" -sort-descending
    • 按存取日期降序排列（最新的在前）

版本歷史

專案包含多個版本檔案，記錄了開發過程中的不同階段：

  • v1: 基本過濾功能
  • v2: 改善參數可讀性
  • v3: 失敗版本
  • v4: 參數化改進
  • v5: Python 和 PowerShell 檔案支援
  • v6: 日期格式從年份開始

系統需求

  • Windows 作業系統
  • Everything 搜尋工具（建議安裝在 C:\Program Files\Everything\）
  • 批次檔案執行權限

注意事項

  1. 確保 Everything 已正確安裝並可執行
  2. 搜尋路徑需要存在且可存取
  3. 日期格式使用 YYYY/MM/DD 格式
  4. 正規表達式中的特殊字元需要適當轉義

自訂配置

您可以根據需要修改以下設定：

  • 搜尋路徑：修改 SEARCH_PATH 變數
  • 檔案模式：修改 SEARCH_PATTERN 變數
  • 日期範圍：修改 DATE_RANGE 變數
  • 排除目錄：修改 SEARCH_exclude 變數

故障排除

如果遇到問題，請檢查：

  1. Everything 是否正確安裝
  2. 搜尋路徑是否存在
  3. 日期格式是否正確
  4. 正規表達式語法是否正確
