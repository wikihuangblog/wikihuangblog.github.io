參考資料搜索工具 (Reference Search Tool)

這是一個基於 FileLocator Pro 的自動化搜索工具，用於根據資料夾名稱中的關鍵字搜索相關的參考資料。

功能說明

  • 自動從當前資料夾路徑中提取關鍵字
  • 支援多關鍵字搜索（以分號分隔）
  • 使用 FileLocator Pro 進行高效的文件搜索
  • 生成 HTML 格式的搜索結果報告

項目結構

app/
├── app.py                    # 主程序（最新版本）
├── app-v1.py                # 早期版本
├── go.bat                   # 批處理啟動腳本
├── 00_ref/
│   └── filelocator-serch-template.py  # FileLocator 使用範例
├── pic/
│   └── image-20250328070554364.png    # 項目說明圖片
├── issue.markdown           # 項目說明文檔
└── README.md               # 本說明文檔

使用方法

1. 環境需求

  • Python 3.x
  • FileLocator Pro（需安裝在預設路徑）
  • Windows 作業系統

2. 快速開始

  1. 將此工具放置在您的參考資料資料夾中
  2. 雙擊執行 go.bat 或在命令列中執行： 
    go.bat

3. 資料夾命名規則

工具會自動從資料夾路徑中提取搜索關鍵字：

  • 格式: Q[數字]_[關鍵字1];[關鍵字2]
  • 範例: Q01_Digitization;AI → 搜索關鍵字：Digitization 和 AI
  • 範例: Q03_SDLC → 搜索關鍵字：SDLC

4. 搜索配置

在 app.py 中可以修改以下設定：

# FileLocator Pro 安裝路徑
filelocator_path = r"C:\Program Files\Mythicsoft\FileLocator Pro\FileLocatorPro.exe"

# 搜索目錄
search_dir = "E:\\00_bookstore\\airitibooks\\download-ok"

# 搜索文件類型
file_pattern = "*.md;*.markdown;*.txt;"

工作原理

  1. 路徑解析: 從命令行參數獲取當前資料夾路徑
  2. 關鍵字提取: 從父資料夾名稱中提取關鍵字
  3. 正則表達式生成: 將多個關鍵字組合成搜索正則表達式
  4. FileLocator 搜索: 調用 FileLocator Pro 執行搜索
  5. 結果輸出: 生成 search_results.html 文件

搜索結果

搜索完成後會生成 search_results.html 文件，包含：

  • 匹配的文件列表
  • 文件內容預覽
  • 關鍵字高亮顯示

版本差異

  • app.py: 最新版本，支援多關鍵字搜索和正則表達式
  • app-v1.py: 早期版本，功能較為簡單

故障排除

常見問題

  1. FileLocator Pro 未找到

    • 確認 FileLocator Pro 已正確安裝
    • 檢查 filelocator_path 變數是否指向正確的安裝路徑

  2. 搜索結果為空

    • 檢查 search_dir 路徑是否正確
    • 確認搜索目錄中存在符合條件的文件

  3. 編碼問題

    • 確保搜索的文件使用 UTF-8 編碼
    • 檢查 FileLocator Pro 的編碼設定

自定義配置

您可以根據需要修改以下參數：

  • search_dir: 更改搜索目錄
  • file_pattern: 修改搜索的文件類型
  • output_file: 更改輸出文件名
  • 搜索參數（-cex, -s, -ofh 等）
