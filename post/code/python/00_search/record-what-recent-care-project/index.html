專案記錄工具 (Record What Recent Care Project)

這是一個用於記錄最近關注專案資料夾的 Python 工具，可以自動建立捷徑檔案並按月份組織，方便追蹤最近的工作進度。

功能特色

  • 📁 記錄最近關注的專案資料夾路徑
  • 📅 按日期時間自動分類到對應月份資料夾
  • 🔗 自動建立 Windows 捷徑檔案 (.lnk)
  • 📝 將記錄保存到 Markdown 檔案中
  • 🗂️ 自動建立年/月資料夾結構

專案結構

app/
├── app.py              # 主要 Python 程式
├── go.bat              # 批次執行檔案
├── project_records.md  # 專案記錄檔案
├── issue.markdown      # 專案說明文件
├── output/             # 輸出資料夾
│   └── 2025/
│       ├── 02/         # 按月份分類的捷徑檔案
│       ├── 03/
│       └── 05/
└── pic/                # 專案截圖

安裝需求

系統需求

  • Windows 作業系統
  • Python 3.x

Python 套件

pip install pywin32

使用方法

方法一：透過 SendTo 功能（推薦）

  1. 將 go.bat 檔案複製到 Windows SendTo 資料夾：

    • 按 Win + R，輸入 shell:sendto
    • 將 go.bat 複製到該資料夾

  2. 使用方式：

    • 在檔案總管中右鍵點擊任何資料夾
    • 選擇「傳送到」→「go.bat」
    • 系統會自動記錄該資料夾路徑並建立捷徑

方法二：直接執行

# 在命令提示字元中執行
go.bat "C:\path\to\your\project\folder"

方法三：直接執行 Python 程式

python app.py "C:\path\to\your\project\folder"

程式碼說明

主要功能

  1. create_shortcut(): 使用 Windows COM 物件建立捷徑檔案
  2. record_folder_path(): 記錄資料夾路徑並建立對應的捷徑檔案
  3. main(): 處理命令列參數並執行主要邏輯

核心邏輯

  1. 接收資料夾路徑作為參數
  2. 記錄當前日期時間和路徑到 project_records.md
  3. 在 output/年/月/ 資料夾中建立對應的捷徑檔案
  4. 自動建立必要的資料夾結構

使用場景

  • 📊 追蹤最近關注的專案進度
  • 🗂️ 快速存取最近使用的專案資料夾
  • 📈 回顧工作歷程和專案發展
  • 🔍 分析工作模式和專案分佈

注意事項

  • 需要 Windows 系統才能建立 .lnk 捷徑檔案
  • 確保有足夠的磁碟空間儲存捷徑檔案
  • 路徑中的特殊字元可能會影響捷徑檔案的建立

技術細節

  • 使用 win32com.client 建立 Windows 捷徑
  • 使用 datetime 模組處理日期時間
  • 使用 os 模組處理檔案路徑和資料夾操作
  • 支援 UTF-8 編碼的 Markdown 檔案

版本資訊

  • 適用系統：Windows 10/11
