crawl-ebook

華藝電子書爬蟲專案

這是一個用於爬取華藝電子書網站書籍資訊的 Python 專案，支援自動化爬取書籍標題、URL 連結、封面圖片和目錄內容。

功能特色

  • 🚀 異步爬取：使用 Playwright 和 asyncio 實現高效異步爬取
  • 📚 多分類支援：支援按分類爬取不同類別的書籍
  • 🖼️ 圖片下載：自動下載書籍封面圖片
  • 📄 目錄爬取：可選功能，爬取書籍目錄並儲存為 HTML
  • 🔄 斷點續傳：支援記錄爬取進度，可從上次中斷處繼續
  • 📁 檔案整理：自動整理下載的檔案到對應資料夾
  • 🔍 重複檢測：自動檢測並處理重複檔案

專案結構

crawl-ebook/
├── app.py                 # 原始爬蟲腳本（舊版本）
├── main.py               # 主程式入口
├── libs/                 # 核心模組
│   ├── BookCrawler.py    # 書籍爬蟲類別
│   ├── Config.py         # 配置管理
│   └── FileManager.py    # 檔案管理
├── 華藝電子書分類/        # 分類配置目錄
│   └── 00_target/        # 目標分類 URL 檔案
├── output/               # 輸出目錄
├── 00_temp-ready-move-area/ # 臨時檔案區域
│   ├── urls/            # URL 檔案
│   └── images/          # 圖片檔案
├── toc_html/            # 目錄 HTML 檔案
└── crawler_config.json  # 爬蟲配置檔案

安裝需求

系統需求

  • Python 3.7+
  • Windows 10/11

依賴套件

pip install playwright aiohttp configparser

安裝 Playwright 瀏覽器

playwright install chromium

使用方法

1. 準備分類 URL 檔案

在 華藝電子書分類/00_target/ 目錄下放置 .url 檔案，每個檔案代表一個書籍分類。

HTML 目錄轉資料夾結構工具

這是一個 Python 應用程式，用於將 HTML 格式的目錄結構轉換為實際的資料夾層次結構，並為每個章節創建 PDF 頁面捷徑。

功能特色

  • 📁 解析 HTML 目錄結構（<ul> 和 <li> 標籤）
  • 🗂️ 自動創建對應的資料夾層次結構
  • 🔢 為每個資料夾添加層次編號前綴（如 01.00.00_）
  • 🔗 為每個章節創建 PDF 頁面捷徑（.lnk 檔案）
  • 🧹 自動清理資料夾名稱，移除 Windows 不允許的字符
  • 📖 支援多層巢狀目錄結構

專案結構

app/
├── app.py                    # 主要應用程式檔案
├── Coding-Interviews.htm     # 輸入的 HTML 目錄檔案
├── folder/                   # 生成的資料夾結構
│   ├── 00_Cover/
│   ├── 01_Chapter-1_-Interview-Process/
│   ├── 02_Chapter-2_-Programming-Languages/
│   └── ...
├── 00_issue/                 # 問題描述
├── 00_ref/                   # 參考檔案
└── README.md                 # 本說明檔案

系統需求

  • Python 3.6+
  • Windows 作業系統（用於創建 .lnk 捷徑）
  • 以下 Python 套件：
    • beautifulsoup4 - HTML 解析
    • pywin32 - Windows COM 介面

安裝步驟

  1. 確保已安裝 Python 3.6 或更高版本
  2. 安裝必要的套件：
pip install beautifulsoup4 pywin32

使用方法

  1. 將您的 HTML 目錄檔案命名為 Coding-Interviews.htm 並放在專案根目錄
  2. 在 app.py 中修改 PDF 檔案路徑（第 115 行）： 
    pdf_path = r"您的PDF檔案路徑"
  3. 執行程式： 
    python app.py
