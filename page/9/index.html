參考資料搜索工具 (Reference Search Tool)

這是一個基於 FileLocator Pro 的自動化搜索工具，用於根據資料夾名稱中的關鍵字搜索相關的參考資料。

功能說明

  • 自動從當前資料夾路徑中提取關鍵字
  • 支援多關鍵字搜索（以分號分隔）
  • 使用 FileLocator Pro 進行高效的文件搜索
  • 生成 HTML 格式的搜索結果報告

項目結構

app/
├── app.py                    # 主程序（最新版本）
├── app-v1.py                # 早期版本
├── go.bat                   # 批處理啟動腳本
├── 00_ref/
│   └── filelocator-serch-template.py  # FileLocator 使用範例
├── pic/
│   └── image-20250328070554364.png    # 項目說明圖片
├── issue.markdown           # 項目說明文檔
└── README.md               # 本說明文檔

使用方法

1. 環境需求

  • Python 3.x
  • FileLocator Pro（需安裝在預設路徑）
  • Windows 作業系統

2. 快速開始

  1. 將此工具放置在您的參考資料資料夾中
  2. 雙擊執行 go.bat 或在命令列中執行： 
    go.bat

MSG 檔案重新命名工具

這是一個用於自動重新命名 Outlook .msg 檔案的 Python 工具，根據郵件的發送日期來為檔案添加日期前綴。

功能特色

  • 🔍 自動從 .msg 檔案中提取發送日期
  • 📅 使用 YYYYMMDD_HHMMSS 格式作為檔案名稱前綴
  • 🔄 避免重複處理已重新命名的檔案
  • 🛡️ 處理檔案名稱衝突（自動添加序號）
  • 📁 批次處理整個資料夾中的所有 .msg 檔案

系統需求

  • Windows 作業系統
  • Microsoft Outlook 已安裝
  • Python 3.6 或更高版本

安裝步驟

  1. 克隆或下載此專案到您的電腦
  2. 安裝必要的依賴套件：
pip install -r requirements.txt

使用方法

  1. 修改 app.py 中的 folder_path 變數，指向包含 .msg 檔案的資料夾：
folder_path = r"D:\your\path\to\msg\files"
  1. 執行程式：
python app.py

HTML 鏈接批量處理工具

這是一個用於批量處理 HTML 文件並自動打開其中鏈接的 Python 工具。

功能描述

此工具會：

  1. 掃描指定資料夾中的所有 HTML 文件
  2. 從每個 HTML 文件中提取第一個符合條件的鏈接（li.classroom-toc-item > a）
  3. 自動在瀏覽器中打開這些鏈接
  4. 根據資料夾名稱計算延遲時間
  5. 處理完成後將 HTML 文件移動到完成資料夾

系統需求

  • Python 3.6+
  • 以下 Python 套件：
    • beautifulsoup4
    • lxml (可選，用於更快的 HTML 解析)

安裝依賴

pip install beautifulsoup4 lxml

使用方法

方法一：使用 Python 腳本

python app.py <目標資料夾路徑>

方法二：使用批次檔

  1. 編輯 go.bat 文件，修改 target_folder 變數為您的目標資料夾路徑
  2. 雙擊執行 go.bat

參數說明

  • target_folder: 包含 HTML 文件的目標資料夾路徑

工作原理

  1. 鏈接提取: 從每個 HTML 文件中查找 li.classroom-toc-item > a 選擇器對應的鏈接
  2. URL 處理: 提取鏈接的基礎 URL（去除文件名部分）
  3. 延遲計算: 根據資料夾名稱中的數字範圍計算延遲時間
    • 例如：資料夾名為 0-9，則延遲時間 = 4秒 × 5 + 5秒 = 25秒
  4. 自動執行: 按順序打開每個鏈接，等待指定時間後移動文件到完成資料夾
