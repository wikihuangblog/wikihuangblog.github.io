視窗切換工具 (Window Switch Tool)

這是一個使用 AutoHotkey 腳本開發的視窗切換工具，可以快速在不同視窗之間切換並執行特定操作。

功能說明

  • 快速視窗切換: 按下 F3 鍵即可快速切換視窗
  • 自動化操作: 切換視窗後自動執行 F1 鍵操作
  • 延遲控制: 內建適當的延遲時間確保操作穩定性

檔案結構

app/
├── switch-window.ahk          # 主要的 AutoHotkey 腳本檔案
├── 00_version/               # 版本控制目錄
│   └── app___original___.lnk # 原始應用程式連結
└── README.md                 # 說明文件

使用方法

前置需求

  1. 安裝 AutoHotkey 軟體
  2. 確保系統已安裝並可正常運行 AutoHotkey

執行步驟

  1. 下載或複製 switch-window.ahk 檔案
  2. 雙擊執行該檔案，或右鍵選擇「Compile Script」編譯為 .exe 檔案
  3. 腳本執行後，按下 F3 鍵即可觸發視窗切換功能

toko-ocr

TOKO OCR 圖片文字識別工具

這是一個基於 Python 和 Tesseract OCR 的圖片文字識別工具，可以將圖片中的文字提取出來並轉換為 Markdown 格式。

功能特色

  • 支援多種圖片格式（JPG、PNG 等）
  • 使用 Tesseract OCR 引擎進行文字識別
  • 支援繁體中文識別
  • 自動將識別結果輸出為 Markdown 格式
  • 批次處理資料夾中的所有圖片

系統需求

  • Python 3.6+
  • Tesseract OCR
  • 相關 Python 套件

安裝步驟

1. 安裝 Tesseract OCR

Windows

  1. 下載並安裝 Tesseract OCR
  2. 預設安裝路徑：C:\Program Files\Tesseract-OCR\tesseract.exe

macOS

brew install tesseract

Ubuntu/Debian

sudo apt-get install tesseract-ocr

2. 安裝 Python 套件

pip install pytesseract pillow

使用方法

基本用法

python app.py <輸入資料夾路徑>

範例

# 處理 input 資料夾中的圖片
python app.py input

# 處理 v1 資料夾中的圖片
python app.py input/v1

專案結構

toko-ocr/
├── app.py                 # 主程式
├── input/                 # 輸入資料夾
│   ├── input.md          # 輸出結果
│   ├── *.jpg             # 待處理的圖片
│   └── v1/               # 子資料夾
│       ├── input.md      # 輸出結果
│       └── *.jpg         # 待處理的圖片
└── README.md             # 說明文件

程式碼說明

主要類別：ImageOCRProcessor

  • __init__(input_folder, languages, tesseract_cmd): 初始化 OCR 處理器
  • ocr_image(image_path, language): 對單一圖片進行 OCR 識別
  • process_images(): 批次處理資料夾中的所有圖片

設定參數

  • languages: 識別語言，預設為 ['chi_tra']（繁體中文）
  • tesseract_cmd: Tesseract 執行檔路徑

輸出格式

程式會自動在輸入資料夾中生成一個與資料夾同名的 .md 檔案，包含所有圖片的 OCR 識別結果。
