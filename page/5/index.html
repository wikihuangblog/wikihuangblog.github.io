grouped-path-bracket-extract

檔案路徑關鍵字提取與統計工具

這是一個用於從檔案路徑中提取大括號內的關鍵字，並生成關鍵字頻率統計圖表的 Python 工具。

功能特色

  • 🔍 關鍵字提取：從檔案路徑中自動提取大括號 {} 內的關鍵字
  • 📊 頻率統計：統計每個關鍵字的出現頻率
  • 🖼️ 視覺化輸出：生成包含關鍵字頻率表格的圖片檔案
  • 🔄 多關鍵字支援：支援單個檔案路徑中包含多個關鍵字的情況
  • 📝 自動命名：根據提取的關鍵字自動生成輸出檔案名稱

檔案結構

app/
├── app.py                                    # 主程式檔案
├── path-with-keyword-in-bracket.txt         # 輸入檔案（包含檔案路徑列表）
├── path-with-keyword-in-bracket-old.txt     # 舊版輸入檔案
├── todo.md                                  # 待辦事項和問題記錄
├── EaseUS_2024_10_15_10_55_59.jpg          # 範例輸出圖片
└── README.md                                # 說明文件

安裝需求

Python 套件

pip install Pillow

系統需求

  • Python 3.6+
  • Windows 系統（字體路徑設定為 Windows 字體）

html-to-md-to-pdf

HTML 轉 PDF/Markdown 轉換工具

這是一個 Python 工具，可以將 HTML 檔案批量轉換為 PDF 和 Markdown 格式。

功能特色

  • 🔄 批量轉換 HTML 檔案為 PDF 和 Markdown 格式
  • 📁 支援目錄結構處理（最多處理 2 層子目錄）
  • 🎨 自定義 CSS 樣式，優化 PDF 輸出格式
  • 📄 支援 B5 頁面大小，最小邊距設定
  • 🚀 簡單易用的批次執行腳本

系統需求

Python 套件

pip install pdfkit html2text weasyprint markdown

外部工具

  • wkhtmltopdf: 用於 HTML 轉 PDF 轉換
    • 下載地址: https://wkhtmltopdf.org/downloads.html
    • 預設安裝路徑: C:\Program Files\wkhtmltopdf\bin\wkhtmltopdf.exe

檔案結構

app/
├── app.py          # 主要轉換程式（使用 pdfkit）
├── app2md.py       # 替代轉換程式（使用 weasyprint）
├── custom.css      # 自定義 CSS 樣式檔案
├── run.bat         # Windows 批次執行腳本
├── issue.txt       # 專案說明
├── target/         # HTML 檔案輸入目錄
├── target---1/     # 備用輸入目錄
├── target-ori/     # 原始檔案目錄
└── output_pdf/     # 輸出目錄（自動建立）

使用方法

方法一：使用批次腳本（推薦）

# 在 Windows 上直接執行
run.bat

方法二：直接執行 Python 程式

# 使用 app.py（pdfkit 版本）
python app.py target

# 使用 app2md.py（weasyprint 版本）
python app2md.py

程式說明

app.py

  • 使用 pdfkit 和 wkhtmltopdf 進行轉換
  • 支援自定義 CSS 樣式
  • 設定 B5 頁面大小，最小邊距
  • 同時產生 PDF 和 Markdown 檔案

app2md.py

  • 使用 weasyprint 進行轉換
  • 先將 HTML 轉為 Markdown，再轉為 PDF
  • 支援 A4 頁面大小
  • 內建 CSS 樣式

輸出設定

PDF 設定

  • 頁面大小: B5 (app.py) / A4 (app2md.py)
  • 邊距: 最小邊距設定
  • 字體: 自動換行，優化排版
  • 縮放: 可調整縮放比例

Markdown 設定

  • 保留連結和圖片
  • 保留文字格式
  • UTF-8 編碼
