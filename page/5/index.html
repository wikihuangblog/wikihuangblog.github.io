  • 🔄 自動將水平 PDF 分割成垂直 PDF

系統需求

  • Python 3.6 或更高版本

安裝依賴

  3. 處理完成的文件將保存在 target/split_pdfs/ 文件夾中

視窗切換工具 (Window Switch Tool)

這是一個使用 AutoHotkey 腳本開發的視窗切換工具，可以快速在不同視窗之間切換並執行特定操作。

功能說明

  • 快速視窗切換: 按下 F3 鍵即可快速切換視窗
  • 自動化操作: 切換視窗後自動執行 F1 鍵操作
  • 延遲控制: 內建適當的延遲時間確保操作穩定性

檔案結構

app/
├── switch-window.ahk          # 主要的 AutoHotkey 腳本檔案
├── 00_version/               # 版本控制目錄
│   └── app___original___.lnk # 原始應用程式連結
└── README.md                 # 說明文件

使用方法

前置需求

  1. 安裝 AutoHotkey 軟體
  2. 確保系統已安裝並可正常運行 AutoHotkey

執行步驟

  1. 下載或複製 switch-window.ahk 檔案
  2. 雙擊執行該檔案，或右鍵選擇「Compile Script」編譯為 .exe 檔案
  3. 腳本執行後，按下 F3 鍵即可觸發視窗切換功能
